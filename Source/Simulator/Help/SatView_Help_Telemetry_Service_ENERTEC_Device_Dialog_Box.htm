ENERTEC TT&C 3801 Dialog Box

SUMMARY

Saturday, May 09, 2015
Version 1.0

SatView™ can be made compatible with a variety of telemetry data provider products. One of those is the ENERTEC TT&C 3801 device.

How do I...

Configure the ENERTEC TT&C 3801 Device

  1. Open the ENERTEC TT&C Dialog Box if it is not open yet.
  2. Enter the name of the device into the Name edit field.
  3. Add the IP address for the device into the IP Address edit field and adjust the TCP/IP port number in the Port spin box.
    Optional:
    • Change the buffer size and timeout interval through the Buffer and Timeout spin boxes.
    • Add a filter by specifying an Application Process Identifier (APID) and/or Virtual Channel Identifier (VCID).
      Select the corresponding check boxes and adjust the numbers inside the APID and VCID spin boxes.
  4. Adjust the epoch with the documentation of the device manufacturer by adjusting the CDS Epoch spin box.
    For the ENERTEC TT&C 3801 devices this is: January 1, 1958.
  5. Adjust the time error correction by selecting the Error check box and inserting the necessary time interval.
    This amount of microseconds is used to adhere with the interval caused by the transmission delay from the spacecraft to the ground station (including delays caused by the ground segment). A positive value is subtracted from the time delivered by the ENERTEC TT&C 3801 device before time stamping the telemetry data. The opposite happens for a negative value.
    Optional:
    • Mark the Delta check box and insert a time interval to be added to the error correction.
  6. Specify the time interval after that the system clock should be correlated with the ENERTEC TT&C 3801 device. This can be achieved by selecting the Adjust the internal clock at an interval of n minutes check box and by entering the number of minutes n to be used as update interval.
  7. Push the OK button.

Prerequisites:
The privilege 'Changing the telemetry service settings' must be enabled to add, configure or remove available telemetry data provider devices.
