HEX Settings Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

Many aspects of a Hexadecimal Display (HEX) are configurable. Layout, retrieval source and print jobs can be configured and modified as required. Furthermore, it is also possible to define areas of special interest, called Spy Locations, that show selected bits extracted from the telemetry data stream.

How do I...

Modify the Layout

  1. Open the HEX Settings Dialog Box if it is not open yet:
    Activate the HEX display and select the Settings... item on the application button's menu. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Layout page.
  3. Specifying the Appearance:

    Changing the Number of Bytes per Line

    • Inside the Format group box adjust the number in the For unformatted data use... edit field.
      Note:
      This option does not apply to the Dumps display type.

    Changing the Color of the Data

    1. Specify the kind of data those color should be changed by selecting it in the For kind of data use... combo box.
      Note:
      This option does not apply to the Dumps display type.
    2. Select the desired color in the corresponding ...color combo box.

    Enumerating Lines and Columns

    • Mark either the Enumerate lines, or Enumerate columns check box or both depending on the kind of enumeration required and adjust the numbers in the corresponding edit fields.
      Note:
      This option does not apply to the Dumps display type.

    Changing the Fonts

    1. Inside the Fonts group box select the Title, Data, Spy Header or Spy Locations radio button depending on which font should be changed. The Title radio button refers to the title bar on top of each display, the Data to the displayed information and the Spy Header/Locations to the locations of special interest, when defined, at the bottom part of the display.
    2. Select the font name in the Name combo box and select the desired style and size in the Style combo box and Size edit field respectively. A sample of the chosen font is always displayed in the Sample area.
    Controlling the Operating Flags:

    Bad Data Flag

    Every received telemetry data packet (or format) has an associated data quality flag. By default, a packet with a bad quality flag is not displayed.

    • Mark the Show bad data check box if the telemetry data packet (or format) should be displayed even if the data quality flag is set to bad.
  4. Push the OK button.

Use Spy Locations

  1. Open the HEX Settings Dialog Box if it is not open yet:
    Activate the HEX display and select the Settings... item on the application button's menu. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Contents page.

  3. Adding a new Spy Location

    1. Specify the radix with which the value at the spy location should be displayed by checking the appropriate radio button:

      Radix

      Description

      Decimal

      The value composed of the bits found at the spy location is displayed with a decimal radix. The first bit is always interpreted as the most significant bit.

      Binary

      The extracted value is displayed with a binary radix.

      Octal

      The extracted value is displayed with a octal radix.

      Hexadecimal

      The extracted value is displayed with a hexadecimal radix.
    2. Specify the position of the spy location by filling in the Byte, Bit and Length edit field.
      Valid ranges for the various edit fields are:

      Field

      Valid Range

      Byte

      >= 0

      Bit

      0...7

      Length

      1...32
      Note:
      The value of a spy location is not updated if its position is (partly) outside the telemetry data packet (or format).
    3. Enter the name of the spy location into the Description edit field.
    4. Determine the color x with which the parts of the spy location should be displayed by selecting the appropriate options in the For part of spy location use color x combo boxes.
    5. Push the Add button.
    Note:

    This option does not apply to the Dumps display type.

    Modifying a Spy Location

    1. Select the spy location to be modified in the Spy Locations list box and perform the required changes.
    2. Push the Modify button.
    Note:

    This option does not apply to the Dumps display type.

    Deleting a Spy Location

    1. Select the spy location to be deleted in the Spy Locations list box.
    2. Push the Remove button in order to delete the selected spy location. Use the Remove All button to delete all at once.
    Note:

    This option does not apply to the Dumps display type.
  4. Push the OK button.

Specify the Retrieval Source

  1. Open the HEX Settings Dialog Box if it is not open yet:
    Activate the HEX and select the Settings... item on the application button's menu. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Retrieving page.
  3. Select the retrieval file to become the source for any retrieval requests by highlighting it in the Data Source combo box.
    Push the Inspect button to check a potential retrieval file. The details about the inspected retrieval file are shown in the Source Information group box.
    Note:
    The set of retrieval files listed in the Data Source combo box can be specified in the Retrievals page of the System Settings Dialog Box.
  4. Push the OK button.

Use Print Job Profiles

  1. Open the HEX Settings Dialog Box if it is not open yet:
    Activate the HEX display and select the Settings... item on the application button's menu. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Printing page.

  3. Creating a new Print Job Profile

    1. Type the name of the print job profile to be created into the Name combo box.
      Note:
      The Default print job profile is used when printing with the quick printing method and can only be modified but not removed.
    2. In the Properties group box specify the print profile to be used with this print job profile by selecting the corresponding profile in the Print Profile combo box. The print profile specifies the printer and paper related properties whereas the print job profile adds some, more HEX display specific, printing directives.
    3. Specify how many lines of the telemetry data or of the spy locations should be printed by checking either the Print all lines or Print the first/last n lines radio button and by filling-in the corresponding edit fields.
      Note:
      Mark the Print Spy Locations check box if the spy locations should be printed as well. Printing restrictions for spy locations can only be specified when this option is checked.
    4. Type a comment into the Text edit field if additional information should be added to the title of the print-out. Mark the Clear after use check box if this text should be used in a single print-out only.
    5. Push the Add button.

    Modifying a Print Job Profile

    • Select the name of the print job profile to be modified in the Name combo box and perform the necessary changes (see 'Creating a new Print Job Profile').

    Deleting a Print Job Profile

    1. Select the name of the print job profile to be deleted in the Name combo box.
    2. Push the Remove button to remove the selected print job profile. Use the Remove All button to delete all existing profiles.
      Note:
      The Default print job profile cannot be deleted.
  4. Push the OK button.

Schedule Print Jobs

  1. Open the HEX Settings Dialog Box if it is not open yet:
    Activate the HEX display and select the Settings... item on the application button's menu. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Jobs page.
  3. Select the print job to be scheduled in the Defined list box.
  4. Specify when the print job should be executed by marking one or more of the available options:

    Option

    Description

    Print at YYYY.DDD.hh:mm:ss

    Prints the HEX display at a specified time.
    Type the time into the associated time edit field.

    Print at hh:mm:ss every day

    Prints the HEX display every day at a specified time.
    Type the time into the associated time edit field.

    Print at name event

    Prints the HEX display when the event name is in a signaled state.
    Select the event in the associated combo box.
  5. When printing at a certain time is enabled it is possible to specify how many times the printing should be repeated and with what interval. In the For repetition type combo box select the repetition characteristics for either daily or non-daily printing. In the case of non-daily printing specify the interval in minutes after which another printing should occur by checking the Repeat every n minutes and by typing the interval n into the associated edit field.
    For both kinds of printing stop conditions can be specified by selecting either the Stop after n copies or Stop at time YYYY.DDD.mm:hh:ss radio button and by filling in the appropriate data into the edit fields.
  6. Push the > button to activate the print job. The name of the print job is copied from the Defined list box into the Active one in order to indicate the activation.
    Whenever a print job needs to be deactivated but not deleted select it in the Active list box and then push the < button.
    Note:
    Deactivated print jobs are not scheduled.
  7. Push the OK button.
