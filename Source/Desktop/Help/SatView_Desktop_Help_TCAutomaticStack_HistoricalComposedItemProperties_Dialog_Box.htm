TC Automatic Stack Telecommand Sequence Properties Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

The TC Automatic Stack provides the necessary support to view details about released telecommand sequences including the encoded raw data transmitted.

How do I...

View the Properties of a released Telecommand Sequence

  1. Open the TC Automatic Stack Telecommand Sequence Properties Dialog Box if it is not open yet:
    Activate the TC Automatic Stack, select the telecommand sequence those properties need to be viewed inside the Historical Pane of either the Normal Stack or the Time-Tagged Stack and select the Edit è Properties... item on the application button's menu. Alternatively, push the Properties button on the display's toolbar.
  2. Select the telecommand of the sequence those details need to be examined in the Entry combo box and check for the requested telecommand properties.
    Notes:
    • Double-click onto one of the parameters in the list box inside the Parameters group box to view its associated calibration table (if any).
    • Examine the data inside the Raw Data group box when looking for the encoded telecommand data.
  3. Push the OK button.

Prerequisites:
The privileges 'Browsing the contents of the automatic telecommand stack' must be enabled to view the properties of released telecommand sequences.
