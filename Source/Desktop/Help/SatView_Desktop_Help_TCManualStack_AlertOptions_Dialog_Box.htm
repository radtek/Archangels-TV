TC Manual Stack Alert Options Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

The TC Manual Stack can issue messages of different severity concerning telecommands or interlocks. This dialog box allows to specify how those messages should be handled.

How do I...

Configure the Telecommand & Interlock Alert Options

  1. Open the TC Manual Stack Settings Dialog Box if it is not open yet:
    Activate the TC Manual Stack and select the Settings... item on the application button's menu. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Operations page.
  3. Push the Alerts... button. This opens the TC Manual Stack Alert Options Dialog Box.
  4. Select the type of incident that should be alerted inside the Type combo box:

    Incident

    Description

    Telecommand release failures

    Covers all situations where one or several telecommands cannot be released due to any reason.

    Interlock failures

    Describes situations where a local interlock switches to a failed state.
  5. Inside the Action group box select the desired audition through the Audition combo box that should be performed when the above selected incident occurs. It will turn up as an event in the Global Eventbox with the color specified in the associated Color combo box. Mark the Blinking & audible check box if the event should also blink (incl. audible sound if chosen) and require an acknowledgment.
  6. Add all notifications to be performed (if any) when an incident of the selected type occurs by choosing them first in the Notifications combo box and then by pressing the Add button.
    Use the Remove button to delete the selected notification profile or Remove All to delete all of them at once.
  7. Push all necessary OK buttons.

Prerequisites:
The privileges 'Changing the contents of the manual telecommand stack' or 'Performing privileged tasks on the manual telecommand stack' must be enabled to configure the alert options.
