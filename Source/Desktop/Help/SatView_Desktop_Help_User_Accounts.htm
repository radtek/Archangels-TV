User Accounts

SUMMARY

Saturday, October 10, 2015
Version 1.0

Each user of SatView™ requires an account describing his own individual rights on the system. Accounts consist of a variable set of privileges and may be limited in time, a suitable feature for guest users. In addition, predefined privilege profiles can simplify the account management and it is possible to manage accounts on remote computers.

How do I...

Create a new User Account

  1. Open the User Accounts Dialog Box if it is not open yet:
    Push the Users button inside the Accounts panel of the System ribbon bar category.
  2. Select the computer on which the new account should be created in the Computer combo box.
    Note:
    User accounts can be created, modified or deleted on remote computers too. Connect to any host not listed in the Computer combo box by typing its name into the associated edit field and by pushing the Connect button.
  3. Specify the spacecraft by selecting it in the Spacecraft combo box.
  4. Enter the user name into the User edit field.
    Note:
    The information about the computer, spacecraft and user builds, together with the assigned set of privileges, an access token i.e. a user account.
  5. Push the Databases... button in order to specify the databases to which this account should have access to. Use the User Accounts Databases Dialog Box to perform this.
  6. Choose one of the predefined privilege profiles by checking either the Administrators, Operators, Users or Guests radio button.
    Push the Change... button if a custom privilege configuration is required and assemble the requested privileges with the User Privileges Dialog Box.
  7. Check the Unique User option if the account should only be used once at a certain time.
    Note:
    This option is used on a RemoteApp™ server environment to ensure that only one user can log-in with that account credentials at a certain time.
  8. Specify the password associated with the account by checking the Use Password check box and by entering the password into the related edit fields or push the None radio button if no password is required.
    Note:
    The Administrator account always requires a password.
  9. Determine how long an account should be valid by checking either the Unlimited or Expires in n Days radio button. Enter the number of days into the associated edit field if the account has a limited validity.
  10. Push the Create button.
    Repeat the previous steps (except for the first one) if more than one account needs to be created.
  11. Push the OK button.

Modify a User Account

  1. Open the User Accounts Dialog Box if it is not open yet:
    Push the Users button inside the Accounts panel of the System ribbon bar category.
  2. Select the computer on which the account should be modified in the Computer combo box.
  3. Select the spacecraft in the Spacecraft combo box.
  4. Select the user name in the User combo box.
  5. Perform the required modifications.
    Note:
    Multiple accounts can be modified before closing this dialog box by repeating the previous three steps.
  6. Push the OK button.

Prerequisites:
The privilege 'Changing the rights of the own account', 'Changing the password of the own account', 'Changing the validity of the own account' or 'Changing the allowed databases of the own account' must be enabled in order to modify the relevant components of the account if 'Creating and modifying (remote) user accounts' is not enabled.

Delete a User Account

  1. Open the User Accounts Dialog Box if it is not open yet:
    Push the Users button inside the Accounts panel of the System ribbon bar category.
  2. Select the computer on which the account should be deleted in the Computer combo box.
    Note:
    User accounts can be created, modified or deleted on remote computers too. Connect to any host not listed in the Computer combo box by typing its name into the associated edit field and by pushing the Connect button.
  3. Specify the spacecraft in the Spacecraft combo box.
  4. Enter the user name in the User combo box.
  5. Push the Delete button.
    Repeat the previous steps (except for the first one) if more than one account needs to be deleted.
    Note:
    The Administrator account cannot be deleted.
  6. Push the OK button.

Prerequisites:
The privilege 'Creating and modifying user accounts' must be enabled in order to create, modify or delete a user account.
'Creating and modifying remote user accounts' is required to do the same on a remote computer.
