Eventbox Open Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

An Eventbox lists events and messages that can be generated due to many possible reasons.
The Global Eventbox has a global scope and lists all real-time events. Normal Eventboxes are available mainly for retrieving historical events.
All events and messages belong to either the System, Spacecraft or User category, depending from where they originate. Their type gives more information about the severity level and is one out of the following list: Success, Informational, Warning, Error or Scheduled or can have a custom type for further differentiation. The Global Eventbox can also be used as a reminder through its capability to schedule events and to combine them with an audition profile and/or a notification profile. Being able to retrieve events even from remote workstations makes the Eventbox a useful tool to investigate problems.

How do I...

Open an Eventbox

  1. Push the Eventbox button inside the Miscellaneous panel of the Displays ribbon bar category.
    This opens the Eventbox Open Dialog Box.
  2. Select the computer those events and messages should be retrieved by choosing its name in the Computer combo box.
    Note:
    Only computers currently available on the LAN are listed. Connect to one not listed there by typing its name into the Computer edit field and by pushing the Connect button.
  3. Choose the spacecraft of interest through the Spacecraft combo box. Mark the Use global event log check box if a global (unique) spacecraft event log is available and should be used instead of the local one.
  4. Select the user by choosing its name in the User combo box.
  5. Choose the operating mode by either checking the Realtime or History radio button. Follow the instructions about how to retrieve with an Eventbox when requesting historical events.
    Note:
    Local Eventboxes can only operate in real-time mode when the Use global event log option is enabled.
  6. Optionally configure the Eventbox.
  7. Push the OK button.

Prerequisites:
The privilege 'Using eventbox display windows' must be enabled.
