Eventbox Setting Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

Many aspects of an Eventbox are configurable. Layout, event filtering and print jobs can be modified as required.

How do I...

Modify the Layout

  1. Open the Eventbox Settings Dialog Box if it is not open yet:
    Activate the Eventbox and select the Settings... item on the application button's menu. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Layout page.
  3. Specifying the Appearance:

    Changing the Columns

    • Inside the Format group box mark columns that should be displayed:

      Column

      Description

      Time

      Shows the time at which the event occurred or the message was displayed.

      Category

      Indicates the category of the event or message.

      Type

      Indicates the type.

      Message

      Displays the text associated with the event or message.

      Comments

      Displays any attached comment.

      Recipients

      Lists all recipients of an event or message.

      User

      Shows the name of the user that was logged-in when the event occurred or the message was displayed.

    Changing the Fonts

    1. Inside the Fonts group box select the Title Items or List Items radio button depending on which font should be changed. The Title Items radio button refers to title bar on top of each display, the List Items to the items listed.
    2. Select the font name in the Name combo box and select the desired style and size in the Style combo box and Size edit field respectively. A sample of the chosen font is always displayed in the Sample area.
    Using a standard Setup:

    Defining a default Layout

    1. Specify the columns to be displayed and the fonts to be used as described above.
    2. Mark the Use as Default check box.
  4. Push the OK button.

Change the Filtering Options

  1. Open the Eventbox Settings Dialog Box if it is not open yet:
    Activate the Eventbox and select the Settings... item on the application button's menu. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Filtering page.

  3. Filtering by Category

    • Mark the System, Spacecraft and/or User check box if events belonging to that category should be displayed.
    Note:

    This option has an effect on future events only.

    Filtering by Type or Subtype

    1. In the Types list box add all event types that should be listed.
      Note:
      If all filters are removed, the standard types 'Success', 'Informational', 'Warning', 'Error' and 'Scheduled' are filtered.
    2. In the Subtypes list box add all event sub-types that should be listed.
      Note:
      If no sub-types are added, no filtering with respect to the sub-type will take place.
    Note:

    This option has an effect on future events only.

    Filtering by Time

    • Mark the Limited from t1 to t2 check box if only events within a certain time period should be displayed and fill the appropriate time values t1, t2 into the associated edit fields.
    Note:

    This option has an effect on future events only.

    Limiting the Number of Events

    • Mark the Limited to maximal n events check box if maximal n events should be displayed and enter the number n into the associated edit field.
    Note:

    The number of events will be adjusted the next time an event occurs.

    Changing the Sort Order

    • Mark the List the latest event as the first one check box if the latest event should always be listed at the top.

    Changing the Behaviors

    At Startup:

    • Automatically load the last Events

      • Mark the At startup load the last n events check box if the last n events should be loaded automatically at startup and enter the number n into the associated edit field.

    At Event Occurrence:

    • Acknowledge an Event automatically

      • Mark the Autoacknowledge check box if every event should be acknowledged automatically (no blinking and no sound).
      Note:

      This option is available in real-time mode only.

      Pop-up the Eventbox

      • Mark the Bring window to top at an event occurrence check box if the window should pop-up when an event occurs.
      Note:

      This option is available in real-time mode only.

      Send a Notification

      • Mark the Send Notification check box if the Global Eventbox should be notified by a message when an event occurs in another Eventbox.
      Note:

      This option is available for local Eventboxes operating in real-time mode only.

    Changing the Colors

    1. Select the category and type of those events that should be displayed in a different color through the Category and Type combo box.
    2. Choose the desired color in the Color combo box.
    Note:

    This option has an effect on future events only.

    Using Sounds

    • Mark the Use Sound check box if events should produce audible sound when attached to an audition profile.
    Note:

    This option has no effect when the autoacknowledge option (see 'Changing the Behaviors') is enabled and is available for the Global Eventbox only.
  4. Push the OK button.

Use Print Job Profiles

  1. Open the Eventbox Settings Dialog Box if it is not open yet:
    Activate the Eventbox and select the Settings... item on the application button's menu. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Printing page.

  3. Creating a new Print Job Profile

    1. Type the name of the print job profile to be created into the Name combo box.
      Note:
      The Default print job profile is used when printing with the quick printing method and can only be modified but not removed.
    2. In the Properties group box specify the print profile to be used with this print job profile by selecting the corresponding profile in the Print Profile combo box. The print profile specifies the printer and paper related properties whereas the print job profile adds some, more Eventbox specific, printing directives.
    3. Select one of the available printing options by checking the corresponding radio button:

      Option

      Description

      Print all events

      Prints all events listed in the Eventbox.

      Print the first/last n events

      Prints the first or last n events.
      Type 'first' or 'last' and the number n into the corresponding edit fields.

      Print the events between t1 and t2

      Prints all events between t1 and t2 only.
      Type t1 and t2 into the corresponding edit fields.
    4. Type a comment into the Text edit field if additional information should be added to the title of the print-out. Mark the Clear after use check box if this text should be used in a single print-out only.
    5. Push the Add button.

    Modifying a Print Job Profile

    • Select the name of the print job profile to be modified in the Name combo box and perform the necessary changes (see 'Creating a new Print Job Profile').

    Deleting a Print Job Profile

    1. Select the name of the print job profile to be deleted in the Name combo box.
    2. Push the Remove button to remove the selected print job profile. Use the Remove All button to delete all existing profiles.
      Note:
      The Default print job profile cannot be deleted.
  4. Push the OK button.

Schedule Print Jobs

  1. Open the Eventbox Settings Dialog Box if it is not open yet:
    Activate the Eventbox and select the Settings... item on the application button's menu. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Jobs page.
  3. Select the print job to be scheduled in the Defined list box.
  4. Specify when the print job should be executed by marking one or more of the available options:

    Option

    Description

    Print at YYYY.DDD.hh:mm:ss

    Prints the Eventbox at a specified time.
    Type the time into the associated time edit field.

    Print at hh:mm:ss every day

    Prints the Eventbox every day at a specified time.
    Type the time into the associated time edit field.

    Print when n events are listed

    Prints the Eventbox as soon as the number n of events is reached.
    Type the number n into the associated edit field.
  5. When printing at a certain time is enabled it is possible to specify how many times the printing should be repeated and with what interval. In the For repetition type combo box select the repetition characteristics for either daily or non-daily printing. In the case of non-daily printing specify the interval in minutes after which another printing should occur by checking the Repeat every n minutes and by typing the interval n into the associated edit field.
    For both kinds of printing stop conditions can be specified by selecting either the Stop after n copies or Stop at time YYYY.DDD.mm:hh:ss radio button and by filling in the appropriate data into the edit fields.
  6. Push the > button to activate the print job. The name of the print job is copied from the Defined list box into the Active one in order to indicate the activation.
    Whenever a print job needs to be deactivated but not deleted select it in the Active list box and then push the < button.
    Note:
    Deactivated print jobs are not scheduled.
  7. Push the OK button.

Prerequisites:
The privilege 'Changing the settings of the global eventbox display window' must be enabled in order to configure the Global Eventbox.
