User Address Book Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ maintains a user address book containing information about the availability (location and time schedule) of the registered persons. This is important for various services that may send reports, messages or documents to those users.

How do I...

Add a new User Address

  1. Open the User Address Book Dialog Box if it is not open yet:
    Push the Users button inside the Address Books panel of the Tools ribbon bar category.
  2. Choose the computer those user address book needs to be updated by selecting it in the Computer combo box or by typing its name into the corresponding edit field and pushing the Connect button.
    Note:
    The 'Changing the contents of a remote user address book' user privilege must be enabled in order to change the contents of a user address book on another computer.
  3. In the Personal page enter the name and first name into the Name and First Name edit fields.
    All other personal information is not mandatory.
    Note:
    Push the Clear button in order to wipe out all fields. This is useful when adding several users with different address details.
  4. Fill-in all relevant information into the Office, Mission, Home and Holiday pages.
    Note:
    Make sure that all address information that is required by the services is supplied correctly (e.g. if SatView™ generates reports that should be sent to a user by mail the corresponding e-mail address must be supplied correctly).
  5. Select the Planner page if a schedule should be attached to the new user and insert the relevant information:
    • Mark the At Home check box if the user can be reached at home as specified on the Home page and enter the time period when the user should be contacted at home by adjusting the Usually from t1 to t2 edit fields and by marking, if desired, the On Working Days only check box.
    • Mark the On Mission check box if the user is on mission and can be reached at the location specified on the Mission page and enter the mission time period into the Leaving on t1 until t2 edit fields.
    • Mark the On Holiday check box if the user is on holiday and can be reached at the location specified on the Holiday page and enter the holiday time period into the Leaving on t1 until t2 edit fields.
  6. Push the Add button.
  7. Push the OK button.

Prerequisites:
The privilege 'Changing the contents of the (a remote) user address book' must be enabled.

Modify a User Address

  1. Open the User Address Book Dialog Box if it is not open yet:
    Push the Users button inside the Address Book panel of the Tools ribbon bar category.
  2. Choose the computer those user address book needs to be updated by selecting it in the Computer combo box or by typing its name into the corresponding edit field and pushing the Connect button.
    Note:
    The 'Changing the contents of a remote user address book' user privilege must be enabled in order to change the contents of a user address book on another computer.
  3. Select the user those address details need to be modified in the Users list box and:
    • Do the necessary changes on the Personal, Office, Mission, Home, Holiday and Planner pages.
      Note:
      Make sure that all address information that is required by the services is supplied correctly (e.g. if SatView™ generates reports that should be sent to a user by mail the corresponding e-mail address must be supplied correctly).
    Note:
    For more details about all the configuration options read how to add a user address (see above).
  4. Push the OK button.

Prerequisites:
The privilege 'Changing the contents of the (a remote) user address book' must be enabled.

Delete a User Address

  1. Open the User Address Book Dialog Box if it is not open yet:
    Push the Users button inside the Address Book panel of the Tools ribbon bar category.
  2. Choose the computer those user address book needs to be updated by selecting it in the Computer combo box or by typing its name into the corresponding edit field and pushing the Connect button.
    Note:
    The 'Changing the contents of a remote user address book' user privilege must be enabled in order to change the contents of a user address book on another computer.
  3. Select the name of the user to be deleted in the Users list box.
  4. Push the Remove button.
  5. Push the OK button.

Prerequisites:
The privilege 'Changing the contents of the (a remote) user address book' must be enabled.

Import User Addresses

  1. Open the User Address Book Dialog Box if it is not open yet and then push the Import... button.
  2. Select the computer from which the user addresses should be imported in the Computer combo box.
    Note:
    Enter a name into the Computer edit field if the requested computer is not listed and push the Connect button.
  3. Select the users to be imported in the Contents list box.
    Note:
    Multiple selections are allowed.
  4. Push the OK button.

Prerequisites:
The privilege 'Changing the contents of the (a remote) user address book' must be enabled.

Print User Addresses

  1. Select the Print User Address Book... item on the application button's menu.
  2. Fill-in all information into the Print Dialog Box.
    Note:
    Use the Page Setup Dialog Box in order to change the current print page settings.
  3. Push the OK button.
