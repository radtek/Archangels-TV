TC Automatic Stack Save Telecommand Template Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

The TC Automatic Stack can handle templates, a convenient way to store and load a telecommand in multiple different configurations. This is yet another way to accelerate the re-use of telecommands.

How do I...

Create a Telecommand Template

  1. Open the TC Automatic Stack Telecommand Properties Dialog Box if it is not open yet:
    Activate the TC Automatic Stack, select the telecommand those properties need to be viewed or modified inside the Scheduled Pane of either the Normal Stack or the Time-Tagged Stack and select the Edit è Properties... item on the application button's menu. Alternatively, push the Properties button on the display's toolbar.
    Note:
    Dispatched telecommands can neither be browsed nor modified.
  2. Push the Save Template... button. This finally opens the TC Automatic Stack Save Telecommand Template Dialog Box.
  3. Type a unique name for the new template to be saved into the Name combo box.
    Note:
    The Related Templates list box can be used to browse all existing templates for the current telecommand.
  4. Add a short description concerning the use of the template into the Description edit field.
  5. Push all necessary OK buttons.

Prerequisites:
The command source must be in a 'SUSPENDED' commanding state to perform the above steps.

Prerequisites:
The privileges 'Changing the contents of the automatic telecommand stack' or 'Performing privileged tasks on the automatic telecommand stack' must be enabled to create, modify or delete templates.

Delete a Telecommand Template

  1. Open the TC Automatic Stack Telecommand Properties Dialog Box if it is not open yet:
    Activate the TC Automatic Stack, select the telecommand those properties need to be viewed or modified inside the Scheduled Pane of either the Normal Stack or the Time-Tagged Stack and select the Edit è Properties... item on the application button's menu. Alternatively, push the Properties button on the display's toolbar.
    Note:
    Dispatched telecommands can neither be browsed nor modified.
  2. Push the Save Template... button. This finally opens the TC Automatic Stack Save Telecommand Template Dialog Box.
  3. Type the name of the template to be deleted into the Name edit field or select the template in the upper list box.
  4. Push the > button to remove the selected template. The >> button deletes all available templates for the telecommand.
  5. Push all necessary OK buttons.

Prerequisites:
The command source must be in a 'SUSPENDED' commanding state to perform the above steps.

Prerequisites:
The privileges 'Changing the contents of the automatic telecommand stack' or 'Performing privileged tasks on the automatic telecommand stack' must be enabled to create, modify or delete templates.
