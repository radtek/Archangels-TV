Miscellaneous

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ provides a convenient way to synchronize the configuration of workstations. This is done by importing and exporting configuration profiles. It is also possible to update the local configuration database with the current settings which prevents a loss of information if the application is not gracefully terminated after a change has taken place.

How do I...

Import a Configuration Profile

  1. Open the System Settings Dialog Box if it is not open yet:
    Push the Settings button inside the Configuration panel of the System ribbon bar category.
  2. Select the Miscellaneous page.
  3. Mark the check boxes of those system components that should be synchronized by the imported configuration profile.
  4. Select the Import As check box and enter the path name of the file containing the configuration data into the associated edit field.
  5. Push the Import button.
    Note:
    Check the Results list box to see if all requested data was imported successfully.
  6. Push the OK button.

Export a Configuration Profile

  1. Open the System Settings Dialog Box if it is not open yet:
    Push the Settings button inside the Configuration panel of the System ribbon bar category.
  2. Select the Miscellaneous page.
  3. Mark the check boxes of those system components that should be exported to a configuration profile.
  4. Select the Export As check box and enter the path name of the file that should contain the exported configuration data into the associated edit field.
  5. Push the Export button.
    Note:
    Check the Results list box to see if all requested data was exported successfully.
  6. Push the OK button.

Update the Configuration Database

  1. Open the System Settings Dialog Box if it is not open yet:
    Push the Settings button inside the Configuration panel of the System ribbon bar category.
  2. Select the Miscellaneous page.
  3. Mark the check boxes of those system components that should be updated in the configuration database.
  4. Push the Update button.
    Note:
    Check the Results list box to see if all requested data was updated successfully.
  5. Push the OK button.

Prerequisites:
The privilege 'Changing and modifying miscellaneous settings' must be enabled in order to synchronize or update configuration settings.
