HEX Open Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

A Hexadecimal Display (HEX) is capable to monitor raw (unprocessed) telemetry data as well as Memory Dumps. In addition to that, so called Spy Locations extract selected portions of the data for further investigation.

How do I...

Open a GRD

  1. Push the HEX button inside the Telemetry Displays panel of the Displays ribbon bar category.
    This opens the HEX Open Dialog Box.
  2. Choose between the available display types by checking the appropriate radio button in the dialog box:

    Type

    Description

    Normal

    Shows the raw data belonging to a specified telemetry packet.
    Select the format to be viewed in the Display combo box.
    Note:
    When monitoring telemetry data that is not conform with the CCSDS Packet Telemetry Standard this option displays the complete telemetry formats generated by the spacecraft.

    Stream

    Shows the raw data of every telemetry packet (or format) received.
    Note:
    This option is not applicable when the telemetry data is not compatible with the CCSDS Packet Telemetry Standard.

    Dumps

    Displays the content of a specified memory dump.
    Select the memory dump to be viewed in the Display combo box.
  3. Choose the data mode by either checking the Realtime or History radio button.
    Follow the instructions about how to retrieve with a HEX display when requesting historical telemetry data.
  4. Optionally configure the HEX display.
  5. Push the OK button.

Prerequisites:
The privilege 'Using hexadecimal display windows' must be enabled.
