TC On-Board Buffer Queues Settings Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

Many aspects of the TC On-Board Buffer Queues Display are configurable. Layout and the print profile can be modified as required.

How do I...

Modify the Layout

  1. Open the TC On-Board Buffer Queues Settings Dialog Box if it is not open yet:
    Activate the TC On-Board Buffer Queues display and select the Settings... item on the application button's menu when the display is activated. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Layout page.
  3. Specifying the Appearance:

    Creating a new Column

    1. Choose the pane which the changes should apply to in the For Pane combo box.
    2. Inside the Format group box choose the column to be inserted by selecting the corresponding name in the Name combo box.
    3. Click at the position on the column items bar below the Name combo box where the selected column should be inserted. A '*' indicating the selected column appears.
    4. Push the Insert After button.
      Note:
      The No. or Name column respectively must always be the left most one and it cannot be deleted.

    Deleting a Column

    1. Choose the pane which the changes should apply to in the For Pane combo box.
    2. Select the column to be deleted by clicking on it in the column items bar below the Name combo box. A '*' indicating the selected column appears.
    3. Push the Remove button.
      Note:
      The No. or Name column respectively cannot be deleted.

    Specifying the Background Grid & Color

    1. Choose the pane which the changes should apply to in the For Pane combo box.
    2. Mark the Grid check box in order to enable a background grid and select the desired color inside the associated color combo box.

    Changing the Fonts

    1. Choose the pane which the changes should apply to in the For Pane combo box.
    2. Inside the Fonts group box select the Title or Items radio button depending on which font should be changed for the selected pane.
    3. Select the font name in the Name combo box and select the desired style and size in the Style combo box and Size edit field respectively. A sample of the chosen font is always displayed in the Sample area.
    Controlling the Operating Flags:

    Tool-tips

    A small tool-tip containing on-board buffer queue or telecommand related details can be displayed on request. When this option is enabled and the cursor is moved over a part containing additional information, a small hand occurs indicating that a tool-tip is available. Press the right mouse button to make it pop-up. The tool-tip disappears automatically when the mouse button is released.

    • Mark the Tooltips check box in order to enable this feature.
  4. Push the OK button.

Modify the Print Profile

  1. Open the TC On-Board Buffer Queues Settings Dialog Box if it is not open yet:
    Activate the TC On-Board Buffer Queues display and select the Settings... item on the application button's menu when the display is activated. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Printing page.
  3. Select the print profile to be used inside the Print Profile combo box. The print profile specifies the printer and paper related properties.
  4. Inside the Options group box choose which on-board buffer type should be printed by selecting it in the Buffer combo box.
  5. Mark the Summary check box if a list of all buffer queues of the selected type has to be included in the print-out.
  6. Choose the Queue check box to print the contents of a specific on-board buffer queue. The queue must be selected in the associated combo box. When this option is selected, it is possible to restrict the amount of data to be printed by choosing the appropriate option inside the Contents group box:

    Range

    Description

    No. from n1 to n2

    Prints all on-board buffer queue entries between number n1 and n2.

    Execution Time from t1 to t2

    Prints all on-board buffer queue entries with an execution time between t1 and t2.

    Note:
    If several filter options are selected they are combined through a logical 'AND'.

  7. Type a comment into the Text edit field if something should be added to the title of the printout. Mark the Clear after use check box if this text should be used on one printout only.
  8. Push the OK button.

Prerequisites:
The privilege 'Changing the telecommand on-board software services' must be enabled.
