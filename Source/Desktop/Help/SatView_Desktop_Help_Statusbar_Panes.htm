Status Bar Properties

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ displays a variety of information on the status bar like, for example, the current time, CPU load, telemetry data reception & telecommand uplink status as well as archiving related indicators. The various panes can be shown or hidden individually as required.

Contents

Pane

Description

Clock Pane

Displays the current time (in UTC). The time displayed in this pane can be synchronized with the time of the telemetry data stream on request.
The associated tool-tip shows the date.

CPU Performance Pane

Shows the percentage of CPU power that SatView™ requires or receives. It does not display the total CPU load. Nevertheless, if the total CPU load is too high (running average over the last hour), the graph changes its color from blue to orange or red (urgent) respectively. In such a case, either some other running applications should be terminated or, if this is not possible, any unused display windows should be closed in order to decrease the load on the CPU.
The associated tool-tip displays other relevant process information.

Telemetry Data Archiving Pane

Shows the current archiving/backup status i.e. if the archiving/backup of telemetry data is enabled and how much space is available on the corresponding media. If multiple archiving files on different storage medias are in use, SatView™ displays information related to that archive file that has the least amount of free space available. The same also applies to the backup mechanism. The Telemetry Data Archive Settings Dialog Box must be used in order to include an archive file into the list of displayable items for this status bar pane.
The associated tool-tip displays more detailed information about the archiving/backup; including also an estimated date for the next backup (if enabled).

Telemetry Device Pane

Displays the current status of the selected telemetry data provider device and also if SatView™ currently is connected to it. The information displayed in this pane is dependent on the kind of telemetry data provider. Some status information however is common to all such devices:

Status

Description

No Device

No telemetry data provider devices are specified.

Not Ready

The current telemetry data provider device is not available.

Ready

The current telemetry data provider device is connected to the associated equipment but no data is received.

Offline (Disabled)

The active telemetry data provider device is currently disabled.

Offline (Scheduled)

The active telemetry data provider device is scheduled to be offline at this moment.

The associated tool-tip shows more specific information about the current telemetry data provider device.

Telecommand Device Pane

Displays the current status of the selected telecommand data uplink device and also if SatView™ currently is connected to it. The information displayed in this pane is dependent on the kind of telecommand data device. Some status information however is common to all such devices:

Status

Description

No Device

No central command handler is available.

Connected & No Device

The workstation is connected to a central command handler but no telecommand uplink device is available.

Connected & Not Ready

The workstation is connected to a central command handler but the current telecommand uplink device is not ready.

Disconnected & No Device

The connection to the central command handler is interrupted and there is no telecommand uplink device available.

Disconnected & Not Ready

The connection to the central command handler is interrupted and the current telecommand uplink device was not ready at the time when this happened.

Disconnected & Ready

The connection to the central command handler is interrupted and the current telecommand uplink device was ready at the time when this happened.

The associated tool-tip shows more specific information about the current telecommand data uplink device.

Remote Network Pane

This pane is visible only if SatView™ is connected to a remote network. In such a case it shows the name of the connected remote server.
The associated tool-tip shows more specific information about that server and how long the physical connection is active already.

Note:
Every pane of the status bar can be hidden on request. This can be achieved by right-clicking on the status bar and selecting the appropriate menu item of the floating menu popping-up.
