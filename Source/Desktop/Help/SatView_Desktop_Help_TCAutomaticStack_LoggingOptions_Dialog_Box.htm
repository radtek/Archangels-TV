TC Automatic Stack Logging Options Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

The TC Automatic Stack is a command source and configuration changes might need to be logged. This dialog box is designed to specify how such changes have to be logged and how long these log entries have to remain archived.

How do I...

Configure the Command Source Logging Options

  1. Open the TC Automatic Stack Settings Dialog Box if it is not open yet:
    Activate the TC Automatic Stack and select the Settings... item on the application button's menu. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Logging page.
  3. Push the Options... button. The TC Automatic Stack Logging Options Dialog Box will occur.
  4. Type the name for the log file into the Filename edit field. Use the Browse... button in order to locate the file if necessary.
    Note:
    The file name can be identical to those of the other (remote) command sources. This may be interesting if only one single log file should be generated.
  5. Check one of the available file policy options:

    Option

    Description

    Overwrite logs as needed

    Enables the cyclic overwrite mode.
    Note:
    As soon as the log file reaches the specified size or no more space is available on the storage media new logs overwrite the oldest ones.

    Overwrite logs older than n days

    Enables the cyclic overwrite mode.
    Note:
    Logs are overwritten by new ones as soon as they are older than n days. This can happen even before the specified file size is reached.

    Do not overwrite any logs

    Disables the cyclic overwrite mode.
    Note:
    Stops logging when either the specified size is reached or no more space is available on the storage media.
    Optional:
    • Mark the Disabled check box in order to prevent the log file from being updated.
  6. Check one of the available file size options:

    Option

    Description

    Unlimited

    Archives logs as long as space is available on the storage media.

    Limited to n KBytes

    Limits the file size to n KB.
    Optional:
    • Mark the Reset check box in order to reset the log file.
  7. Push all necessary OK buttons.

Prerequisites:
The privileges 'Changing the contents of the automatic telecommand stack' or 'Performing privileged tasks on the automatic telecommand stack' must be enabled to configure the logging options.
