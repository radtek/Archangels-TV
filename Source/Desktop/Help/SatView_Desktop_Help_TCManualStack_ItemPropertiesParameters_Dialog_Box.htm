TC Manual Stack Telecommand Parameters Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

The TC Manual Stack provides full support to configure telecommands or telecommand sequences. This includes the initialization of their contained telecommand parameters. This dialog box simplifies the determination of the correct value for those parameters by showing the attached calibration tables.

How do I...

Browse Telecommand Parameter Details

  1. Open the TC Manual Stack Telecommand Properties Dialog Box or TC Manual Stack Telecommand Sequence Properties Dialog Box if it is not open yet:
    Activate the TC Manual Stack, select the telecommand or telecommand sequence those properties need to be viewed or modified and select the Edit è Properties... item on the application button's menu. Alternatively, push the Properties button on the display's toolbar.
    Note:
    If the telecommand or telecommand sequence is not yet loaded onto the stack proceed as follows:
    1. Open the TC Manual Stack Load Telecommand Dialog Box if it is not open yet:
      Select the Load... item on the application button's menu when the display is activated. Alternatively, push the Load button on the display's toolbar.
    2. Check the TC Functions radio button to load a telecommand or the TC Sequences radio button for telecommand sequences and select the desired item inside the list box below the Filter combo box or type its name into the Load edit field.
    3. Push the Properties... button.
      When a telecommand sequence is loaded, select the required member inside the Entry combo box.
  2. Double-click the parameter those calibration details should be shown inside the list box of the Parameters group box.
    The TC Manual Stack Telecommand Parameters Dialog Box will appear.
    Note:
    This feature is available only for telecommand parameters that have an associated calibration curve.
  3. Select the appropriate calibration curve in the Name combo box inside the Calibration page.
    Note:
    If a predefined calibrated point is appropriate to be used as a value for the parameter, that value can directly be copied to the Value edit field by double-clicking on it.
  4. Push all necessary OK buttons.
