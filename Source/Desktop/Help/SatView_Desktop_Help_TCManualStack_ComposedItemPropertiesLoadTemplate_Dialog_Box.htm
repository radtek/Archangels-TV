TC Manual Stack Load Telecommand Sequence Template Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

The TC Manual Stack can handle templates, a convenient way to store and load a telecommand sequence in multiple different configurations. This is yet another way to accelerate the re-use of telecommand sequences.

How do I...

Load a Telecommand Sequence Template

  1. Open the TC Manual Stack Load Telecommand Dialog Box if it is not open yet:
    Select the Load... item on the application button's menu when the display is activated. Alternatively, push the Load button on the display's toolbar.
  2. Check the TC Sequences radio button.
  3. Select the telecommand sequence to be loaded inside the list box or directly enter its name into the Load edit field.
  4. Push the Properties... button. The TC Manual Stack Telecommand Sequence Properties Dialog Box will appear.
  5. Push the Load Template... button. This finally opens the TC Manual Stack Load Telecommand Sequence Template Dialog Box.
  6. Type the name of the template to be loaded into the Name edit field or select the template in the upper list box.
  7. Push all necessary OK buttons.

Prerequisites:
The privileges 'Changing the contents of the manual telecommand stack' or 'Performing privileged tasks on the manual telecommand stack' must be enabled to configure a telecommand sequence.

Delete a Telecommand Sequence Template

  1. Open the TC Manual Stack Load Telecommand Dialog Box if it is not open yet:
    Select the Load... item on the application button's menu when the display is activated. Alternatively, push the Load button on the display's toolbar.
  2. Check the TC Sequences radio button.
  3. Select the telecommand sequence of which a template should be deleted inside the list box or directly enter its name into the Load edit field.
  4. Push the Properties... button. The TC Manual Stack Telecommand Sequence Properties Dialog Box will appear.
  5. Push the Load Template... button. This finally opens the TC Manual Stack Load Telecommand Sequence Template Dialog Box.
  6. Type the name of the template to be deleted into the Name edit field or select the template in the upper list box.
  7. Push the > button to remove the selected template. The >> button deletes all available templates for the telecommand sequence.
  8. Push all necessary OK buttons.

Prerequisites:
The privileges 'Changing the contents of the manual telecommand stack' or 'Performing privileged tasks on the manual telecommand stack' must be enabled to create, modify or delete templates.
