SatView™ requires an initialization the first time it executes. This registration is performed by the Welcome Dialog Box.

How do I...

Register SatView™

  1. Select the name of the spacecraft to be supported in the Spacecraft combo box.
    Note:
    If the requested spacecraft does not exist, its name can be entered manually. In this case however, an associated database may not be available.
  2. Enter the Administrator password into the Password edit field.
    Note:
    The password must be identical to the Administrator password known at the operating system level.
  3. Push the OK button.

Prerequisites:
An account with administrative privileges is required to perform a successful registration.
