GRD Settings Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

Many aspects of a Graphic Display (GRD) are configurable. Layout, chart properties, parameters, retrieval source and print jobs can be modified as required. Furthermore, it is also possible to define the way how parameter values should be displayed with respect to validity and out-of-limit status colors. Tool-tips, providing details about parameters and, in particular, their values can be enabled or disabled as well.

How do I...

Modify the Layout

  1. Open the GRD Settings Dialog Box if it is not open yet:
    Activate the GRD and select the Settings... item on the application button's menu. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Layout page.
  3. Specifying the Appearance:

    Creating a new Column

    1. Inside the Format group box choose the column to be inserted by selecting the corresponding name in the Name combo box.
    2. Click at the position on the column items bar below the Name combo box where the selected column should be inserted.
      A '*' indicating the selected column appears.
    3. Push the Insert Before or Insert After button.

    Deleting a Column

    1. Select the column to be deleted by clicking on it in the column items bar below the Name combo box.
      A '*' indicating the selected column appears.
    2. Push the Remove button.
      Note:
      The ID column cannot be deleted.

    Changing the Out-of-Limit Colors

    1. Select the out-of-limit status of which the color association should be changed in the For values with status combo box.
    2. Choose the desired color in the ...use color combo box.
    Note:

    Any changes to the out-of-limit status color association apply to the complete graphic display and not to a specific parameter only.

    Changing the Fonts

    1. Inside the Fonts group box select the font that should be changed in the For the area combo box.
    2. Select the font name in the Name combo box and select the desired style and size in the Style combo box and Size edit field respectively. A sample of the chosen font is always displayed in the Sample area.
    Controlling the Operating Flags:

    Invalid Data Flag

    This flag controls the visibility behavior of the telemetry parameter values with respect to their validity.

    • Mark the Invalid Data check box if the parameter values should be displayed even when the associated validity is false.

    Bad Data Flag

    Every telemetry parameter has an associated data quality flag referring to its value. By default, a parameter value with a bad quality flag is not updated.

    • Mark the Bad Data check box if the parameter values should be displayed even if their data quality flag is set to bad.

    Tool-tips

    A small tool-tip containing parameter related details can be displayed on request. When this option is enabled and the cursor is moved over a parameter inside the GRD, a small hand occurs indicating that a tool-tip is available. Press the right mouse button to make it pop-up. The tool-tip disappears automatically when the mouse button is released.

    • Mark the Tooltips check box in order to enable this feature.
  4. Push the OK button.

Change the Chart

  1. Open the GRD Settings Dialog Box if it is not open yet:
    Activate the GRD and select the Settings... item on the application button's menu. Alternatively, push the Settings button on the display's toolbar.
  2. Specifying the Characteristics:

    Choosing the Chart Type

    1. Select the Chart page.
    2. Select one of the chart types by checking the appropriate radio button:

      Type

      Description

      Normal

      Plots the parameter values against time:
      Each parameter contributing to the plot uses the complete vertical extension of the graph.

      Stripchart

      Plots the parameter values against time:
      Each parameter is plotted relative to an assigned horizontal slot. Clipping is performed so that parameter values outside the assigned slots are not visible.

      Stackchart

      Plots the parameter values against time:
      Each parameter is stacked on top of the one beneath it.

      Diagram

      Plots parameters against each other:
      If more than two parameters are contributing to the plot, the value of the first one of each pair defines the x-value, the second one the y-value.

    Changing the Chart Duration

    1. Select the Chart page.
    2. In the Duration group box specify the duration of the graph (i.e. the time until the graph is restarted or overwritten depending on the plotting mode) by filling-in the appropriate numbers into the Days, Hours and Minutes edit fields.
      Specify the update interval of the graph by entering the number of minutes after which the graph should be updated into the Update edit field.
      Note:
      When plotting in the Scroll mode it is possible to specify a visible interval (<= duration of the graph) after which the graph should be scrolled. This can be done by marking the Interval check box and by entering the number of percent (with respect to the duration of the graph) the visible part should consist of. An update interval of zero minutes indicates a default refresh period.

    Choosing the Plot Mode

    1. Select the Chart page.
    2. Select one of the plotting modes by checking the appropriate radio button:

      Mode

      Description

      Normal

      Plots the parameter values as lines.

      Area

      Fills the space between the plotted lines and the x-axis with the color assigned to the corresponding parameter.
      Note:
      This option is not available for the Diagram chart type.

    Changing the Plot Behavior

    1. Select the Chart page.
    2. Select one of the plotting behaviors by checking the appropriate radio button:

      Behavior

      Description

      Scroll

      Scrolls the contents of the graph when the specified visible interval time is expired. The graph is restarted completely when the duration time (>= visible interval) is expired.
      Note:
      This option is not available for the Diagram chart type.

      Overwrite

      Overwrites the graph when the specified duration time is expired. The current location is indicated by a vertical slider.
      Note:
      This option is not available for the Diagram chart type.

      Restart

      Restarts the graph when the specified duration time is expired.

    Changing the Plot Area

    1. Select the Chart page.
    2. The following actions can be performed to change the plot area:

      Action

      How to do it

      Modify the plot area size

      Enter the width and height of the area by pushing the Width and/or Height check box inside the Graph group box and enter the appropriate sizes.
      Note:
      A default width and height is used when no size is specified.

      Modify the plot area grids and ticks

      Specify the number of horizontal and vertical grid lines or ticks respectively by marking the Horizontal and/or Vertical check box. Enter the number of desired grid lines or ticks into the corresponding edit fields. The width of each grid line (or tick) can also be specified by using the spin box on half height between the previously mentioned check boxes.
      The color of the grid lines (ticks) can be chosen with the Color combo box.

      Modify the background color of the plot area

      Choose the color for the background of the plot area by selecting the desired color in the Background combo box inside the Color group box.

      Modify the color of the plot area labels

      Choose the color for the plot area labels by selecting it in the Labels combo box inside the Color group box.
  3. Changing the Contents:

    Adding a Parameter

    1. Select the Parameters page.
    2. Select the parameter to be added by choosing it in the Parameters combo box.
    3. Determine the position where the parameter should be inserted by clicking onto the desired parameter inside the Contents list box after which the new one should be inserted.
    4. Push the Add button.
    5. Determine how the parameter value should be displayed by selecting the appropriate item in the Value combo box inside the Properties group box:

      Coding & Radix

      Description

      Default

      Shows the parameter value in a calibrated form when possible otherwise it is presented in a coded form.

      Decimal

      The raw (not calibrated) value is displayed with a decimal radix.

      Binary

      The raw value is displayed with a binary radix.

      Octal

      The raw value is displayed with an octal radix.

      Hexadecimal

      The raw value is displayed with a hexadecimal radix.

      Inverse

      Displays the one's complement of the parameter value with a binary radix.
    6. If a non-default number of decimal places (for floating-point numbers) should be used to display a parameter's value then mark the Digits check box and enter the required number into the associated spin box.
    7. Mark the Occurrence check box if not all occurrences of a parameter value should be plotted and enter the corresponding number into the related edit field.
    8. Enter a minimum and maximum value to be applied to the plotting of the parameter values into the Minimum and Maximum edit fields.
      Note:
      For parameters with a textual calibration curve all allowed values are listed in the Minimum and Maximum combo boxes.
    9. Type an optional comment related to the parameter into the Comments edit field.
    10. Choose a color for the parameter through the Color combo box.
    11. Specify the characteristics of the line with which the plotted parameter values should be drawn by selecting the appropriate line style, symbol and width in the Style, Symbol and Width combo boxes or edit field respectively.

    Modifying a Parameter

    1. Select the Parameters page.
    2. Select the parameter to be modified by choosing it in the Contents combo box.
      Perform the necessary changes.
    3. Push the Modify button.

    Deleting a Parameter

    1. Select the Parameters page.
    2. Select the parameter to be removed by choosing it in the Contents combo box.
    3. Push the Remove button in order to remove the selected parameter. Using the Remove All button deletes all parameters.
  4. Push the OK button.

Specify the Retrieval Source

  1. Open the GRD Settings Dialog Box if it is not open yet:
    Activate the GRD and select the Settings... item on the application button's menu. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Retrieving page.
  3. Select the retrieval file to become the source for any retrieval requests by highlighting it in the Data Source combo box.
    Push the Inspect button to check a potential retrieval file. The details about the inspected retrieval file are shown in the Source Information group box.
    Note:
    The set of retrieval files listed in the Data Source combo box can be specified in the Retrievals page of the System Settings Dialog Box.
  4. Push the OK button.

Use Print Job Profiles

  1. Open the GRD Settings Dialog Box if it is not open yet:
    Activate the GRD and select the Settings... item on the application button's menu. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Printing page.

  3. Creating a new Print Job Profile

    1. Type the name of the print job profile to be created into the Name combo box.
      Note:
      The Default print job profile is used when printing with the quick printing method and can only be modified but not removed.
    2. In the Properties group box specify the print profile to be used with this print job profile by selecting the corresponding profile in the Print Profile combo box. The print profile specifies the printer and paper related properties whereas the print job profile adds some, more GRD specific, printing directives.
    3. Mark the Print OOL Legend check box if a small legend should be included on the print-out indicating the out-of-limit status color association.
      Note:
      This option is useful when printing in colors.
    4. Mark the Underline OOLs check box if an out-of-limit value should be underlined.
      Note:
      This option is useful when printing without colors.
    5. Mark the No Colors check box if the print-out should be in monochrome despite the printer's color capability.
    6. Determine the scaling of the print-out by checking either the Scale automatically to paper size or Use a fixed scaling factor of n% check box whereby the factor n needs to be filled into the corresponding edit field. Scaling is always done in an isotropic way.
    7. Type a comment into the Text edit field if additional information should be added to the title of the print-out. Mark the Clear after use check box if this text should be used in a single print-out only.
    8. Push the Add button.

    Modifying a Print Job Profile

    • Select the name of the print job profile to be modified in the Name combo box and perform the necessary changes (see 'Creating a new Print Job Profile').

    Deleting a Print Job Profile

    1. Select the name of the print job profile to be deleted in the Name combo box.
    2. Push the Remove button to remove the selected print job profile. Use the Remove All button to delete all existing profiles.
      Note:
      The Default print job profile cannot be deleted.
  4. Push the OK button.

Schedule Print Jobs

  1. Open the GRD Settings Dialog Box if it is not open yet:
    Activate the GRD and select the Settings... item on the application button's menu. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Jobs page.
  3. Select the print job to be scheduled in the Defined list box.
  4. Specify when the print job should be executed by marking one or more of the available options:

    Option

    Description

    Print at YYYY.DDD.hh:mm:ss

    Prints the GRD at a specified time.
    Type the time into the associated time edit field.

    Print at hh:mm:ss every day

    Prints the GRD every day at a specified time.
    Type the time into the associated time edit field.

    Print at name event

    Prints the GRD when the event name is in a signaled state.
    Select the event in the associated combo box.

    Print at expiration time

    Prints the GRD as soon as the chart duration time is reached.
    Mark the Print at expiration time check box.
  5. When printing at a certain time is enabled it is possible to specify how many times the printing should be repeated and with what interval. In the For repetition type combo box select the repetition characteristics for either daily or non-daily printing. In the case of non-daily printing specify the interval in minutes after which another printing should occur by checking the Repeat every n minutes and by typing the interval n into the associated edit field.
    For both kinds of printing stop conditions can be specified by selecting either the Stop after n copies or Stop at time YYYY.DDD.mm:hh:ss radio button and by filling in the appropriate data into the edit fields.
  6. Push the > button to activate the print job. The name of the print job is copied from the Defined list box into the Active one in order to indicate the activation.
    Whenever a print job needs to be deactivated but not deleted select it in the Active list box and then push the < button.
    Note:
    Deactivated print jobs are not scheduled.
  7. Push the OK button.
