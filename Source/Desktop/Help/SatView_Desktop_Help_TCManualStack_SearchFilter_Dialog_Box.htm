TC Manual Stack Telecommand Filter Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

The TC Manual Stack supports filters that can be used to speed-up the search for telecommands or telecommand sequences. This dialog box is designed to provide an interface to create, modify or delete such search filters.

How do I...

Create a new Search Filter

  1. Open the TC Manual Stack Load Telecommand Dialog Box if it is not open yet:
    Select the Load... item on the application button's menu when the display is activated. Alternatively, push the Load button on the display's toolbar.
  2. Check either the TC Functions radio button or the TC Sequences radio button depending on whether a search filter to look for telecommands or telecommand sequences should be created.
  3. Push the Add... button. The TC Manual Stack Telecommand Filter Dialog Box will occur.
    Note:
    An other way to get to this dialog box is by opening the TC Manual Stack Load Telecommand Stacks Dialog Box and then pushing the Add... button.
  4. Type a name for the new filter into the Name edit field.
  5. Specify a search key inside the Key combo box.
  6. Enter a corresponding value to be searched for into the Value edit field.
    Note:
    Multiple values are allowed and any text has to be enclosed in double quotes. Use either '&' (for a logical 'AND') or '|' (for an OR). There is no precedence in the evaluation of an expression; it is just processed from the left to the right.
  7. Push the Add button.
    Repeat the previous steps (except for the first one) if more than one search key should be added.
  8. Push all necessary OK buttons.

Prerequisites:
The privileges 'Changing the contents of the manual telecommand stack' or 'Performing privileged tasks on the manual telecommand stack' must be enabled to create, modify or delete search filters.

Modify a Search Filter

  1. Open the TC Manual Stack Load Telecommand Dialog Box if it is not open yet:
    Select the Load... item on the application button's menu when the display is activated. Alternatively, push the Load button on the display's toolbar.
  2. Check either the TC Functions radio button or the TC Sequences radio button depending on whether a search filter to look for telecommands or telecommand sequences should be modified.
  3. Select the filter to be modified inside the Filter combo box.
  4. Push the Modify... button. The TC Manual Stack Telecommand Filter Dialog Box will occur.
    Note:
    An other way to get to this dialog box is by opening the TC Manual Stack Load Telecommand Stacks Dialog Box and then pushing the Modify... button.
  5. Select the search key to be added or one those value needs to be changed in the Key combo box.
  6. Enter or modify the corresponding value inside the Value edit field.
    Repeat the previous two steps in order to alter more than one search key.
  7. Push all necessary OK buttons.

Prerequisites:
The privileges 'Changing the contents of the manual telecommand stack' or 'Performing privileged tasks on the manual telecommand stack' must be enabled to create, modify or delete search filters.

Delete a Search Filter

  1. Open the TC Manual Stack Load Telecommand Dialog Box if it is not open yet:
    Select the Load... item on the application button's menu when the display is activated. Alternatively, push the Load button on the display's toolbar.
  2. Check either the TC Functions radio button or the TC Sequences radio button depending on whether a search filter to look for telecommands or telecommand sequences should be deleted.
  3. Select the filter to be deleted inside the Filter combo box.
  4. Push the Remove button. Using the Remove All button clears all filters.
  5. Push the OK button.

Prerequisites:
The privileges 'Changing the contents of the manual telecommand stack' or 'Performing privileged tasks on the manual telecommand stack' must be enabled to create, modify or delete search filters.
