AND Parameters Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

Alphanumeric Displays (AND) provide the capability to display additional details about the telemetry parameters they contain.

How do I...

View AND Parameter Details

  1. Open the AND Parameters Dialog Box if it is not open yet by double-clicking on the parameter inside the AND to be viewed or by selecting the Parameters... item on the application menu when the display is activated. Alternatively, push the Telemetry Parameters button on the display's toolbar.
  2. Select the parameter those properties should be shown in the Parameter combo box of the dialog box (if necessary).
  3. Choose the appropriate property page in order to view the desired information.
  4. Push the OK button.
Note:

Additional parameter details can also be obtained through the use of tool-tips: Right-click on one of the parameters to view additional column related information.
