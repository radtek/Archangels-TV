Display Area

SUMMARY

Saturday, October 10, 2015
Version 1.0

The display area hosts the various display windows. It can be configured to contain one or more display pages arranged as tabs with a definable title each and access rights. A setup with multiple display pages for example is preferable on notebooks where screen space is limited.

How do I...

Configure the Display Area

  1. Open the System Settings Dialog Box if it is not open yet:
    Push the Settings button inside the Configuration panel of the System ribbon bar category.
  2. Select the Displayarea page.
  3. Push the Single Display Page or Multiple Display Pages radio button depending on the preferred configuration of the display area.
    Note:
    The multiple display pages configuration creates several stacked display areas.
  4. Push the OK button.

Create a new Display Page

This section is only applicable for a Multiple Display Pages configuration.

  1. Open the System Settings Dialog Box if it is not open yet:
    Push the Settings button inside the Configuration panel of the System ribbon bar category.
  2. Select the Displayarea page.
  3. Enter the (unique) title of the display page into the Title edit field.
  4. Mark the Prevent global display windows from popping-up on this page check box in order to inhibit all global display windows from alerting through popping-up.
  5. Specify the list of users that should be granted access to the display page by checking either the All Users or Limited to the users radio button.
    When access is limited to a set of users, push the Add button in order to specify the individual users. The Accounts Directory Dialog Box where the desired users can be chosen appears. Repeat this step as many times as users need to be added to the access list.
    Every user that has no access rights to the display page must be removed from the access list with the Remove button.
  6. Specify the password required to access the display page by checking one of the following options:

    Option

    Description

    None

    No password is required to access the display page.

    User Password

    The password is identical with the one of the user currently logged-in.

    Special

    Allows the use of a special password specific to this display page and identical for all users. Enter the password into the Special Password and Confirmation edit fields.
  7. Push either the Insert Before or Insert After button depending on where the display page should be inserted.
    Repeat the previous steps if more than one display page should be inserted.
  8. Push the OK button.

Modify a Display Page

This section is only applicable for a Multiple Display Pages configuration.

  1. Open the System Settings Dialog Box if it is not open yet:
    Push the Settings button inside the Configuration panel of the System ribbon bar category.
  2. Select the Displayarea page.
  3. Perform the required modifications with respect to the display page title or access policy.
  4. Push the Modify button.
  5. Push the OK button.

Delete a Display Page

This section is only applicable for a Multiple Display Pages configuration.

  1. Open the System Settings Dialog Box if it is not open yet:
    Push the Settings button inside the Configuration panel of the System ribbon bar category.
  2. Select the Displayarea page.
  3. Select the display page that should be deleted on the tab bar.
  4. Push the Delete button.
    Repeat the previous two steps if more than one display page should be deleted.
  5. Push the OK button.

Note:
At least one display page must exist.

Prerequisites:
The privilege 'Changing the layout of the display area' must be enabled in order to change the characteristics of the display area.
