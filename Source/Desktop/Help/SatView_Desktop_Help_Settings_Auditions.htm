Auditions

SUMMARY

Saturday, October 10, 2015
Version 1.0

Audition profiles are used to summarize audition properties as there are, for example, the kind of audition like wave, voice, beeps and so on, the characteristics of the sound like volume, balance, pitch and other sound related settings. Each audition profile has a unique name and its use greatly simplifies the specification of how an event should be audited.
The Default audition profile describes the default audition behavior. It can be modified but not deleted.

How do I...

Create a new Audition Profile

  1. Open the System Settings Dialog Box if it is not open yet:
    Push the Settings button inside the Configuration panel of the System ribbon bar category.
  2. Select the Auditions page.
  3. Enter the name of the new audition profile into the Name edit field.
    Note:
    The name of the audition profile must be unique.
  4. Choose the sound to be associated with this profile by checking either the None, Speaker, Beep, Wave or Voice radio button:

    Type

    Description

    None

    No sound is played.
    Note:
    No further settings are required for this option.

    Speaker

    Generates a short beep with the speaker.
    Note:
    No further settings are required for this option.

    Beep

    Generates a beep at a specified frequency:

    1. Enter the desired frequency into the associated edit field.
      A range between 200-2000 Hz is applicable.
    2. Select the audio adapter to be used in order to generate this sound in the Use Adapter combo box.
    3. Regulate the volume and balance through the Volume sliders.

    Wave

    Plays a wave file:

    1. Enter the name of the wave file into the Sound File edit field.
      Use the Browse... button to search for such a file.
    2. Select the audio adapter to be used in order to generate this sound in the Use Adapter combo box.
    3. Regulate the volume and balance through the Volume sliders.

    Voice

    Synthesizes a human voice:

    1. Select the gender of the voice in the Gender combo box.
    2. Determine the age of the voice by the Age combo box.
    3. Select one of the available speakers with the selected gender and age in the Speaker combo box.
      If no speaker suiting the specified gender and age can be found a default voice will be used.
    4. Select the audio adapter to be used in order to generate this sound in the Use Adapter combo box.
    5. Regulate the volume, balance, playback rate and pitch through the Volume, Playback Rate and Pitch sliders.
  5. Push the Add button.
  6. Push the OK button.

Modify an Audition Profile

  1. Open the System Settings Dialog Box if it is not open yet:
    Push the Settings button inside the Configuration panel of the System ribbon bar category.
  2. Select the Auditions page.
  3. Select the name of the audition profile to be modified in the Name combo box and perform the required changes.
  4. Push the OK button.

Delete an Audition Profile

  1. Open the System Settings Dialog Box if it is not open yet:
    Push the Settings button inside the Configuration panel of the System ribbon bar category.
  2. Select the Auditions page.
  3. Select the name of the audition profile to be deleted in the Name combo box.
  4. Push the Remove button. Use the Remove All button in order to delete all existing audition profiles.
    Note:
    The Default audition profile cannot be deleted.
  5. Push the OK button.

Prerequisites:
The privilege 'Creating and modifying audition profiles' must be enabled in order to create, modify or delete audition profiles.
