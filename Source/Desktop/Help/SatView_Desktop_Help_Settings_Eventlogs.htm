Event Logs

SUMMARY

Saturday, October 10, 2015
Version 1.0

Event logs are special files used to archive events that occurred and were displayed in the Global Eventbox. For each of the existing event categories System, Spacecraft and User a separate file exists. Special considerations concerning the use of global databases require the existence of an additional file associated with a 'pseudo category', called Global Spacecraft.

How do I...

Specify an Event Log File

  1. Open the System Settings Dialog Box if it is not open yet:
    Push the Settings button inside the Configuration panel of the System ribbon bar category.
  2. Select the Eventlogs page.
  3. Select the user account those event log files are going to be modified by selecting the appropriate computer, spacecraft and user in the associated Computer, Spacecraft and User combo boxes.
    Note:
    Connect to any host not listed by typing its name into the Computer edit field and pushing the Connect button.
  4. Select the category those event log file name needs to be changed by selecting it in the Category combo box.
  5. Enter the new file name into the Filename edit field. Use the Browse button in order to search for an existing file.
    Note:
    If the file is located on a different computer provide the file name in the Universal Naming Conventions (UNC) format.
  6. Repeat the last two steps if the event log files of other categories need to be changed as well.
  7. Push the OK button.

Prerequisites:
The privilege 'Creating and modifying (remote) eventlogs' or 'Changing the filename of the own eventlog' must be enabled in order to modify the (remote) event log file name.

Specify the Event Log Policy

  1. Open the System Settings Dialog Box if it is not open yet:
    Push the Settings button inside the Configuration panel of the System ribbon bar category.
  2. Select the Eventlogs page.
  3. Select the user account those event log files are going to be modified by selecting the appropriate computer, spacecraft and user in the associated Computer, Spacecraft and User combo boxes.
    Note:
    Connect to any host not listed by typing its name into the Computer edit field and pushing the Connect button.
  4. Select the category those event log policy needs to be changed by selecting it in the Category combo box.
  5. Check one of the available file policy options:

    Option

    Description

    Overwrite events as needed

    Enables the cyclic overwrite mode.
    Note:
    As soon as the event log file reaches the specified size or no more space is available on the storage media new events overwrite the oldest ones.

    Overwrite events older than n days

    Enables the cyclic overwrite mode.
    Note:
    Events are overwritten by new ones as soon as they are older than n days. This can happen even before the specified file size is reached.

    Do not overwrite any events

    Disables the cyclic overwrite mode.
  6. Repeat the last two steps if the event log policy of other categories need to be changed as well.
  7. Push the OK button.

Prerequisites:
The privilege 'Creating and modifying (remote) eventlogs' or 'Changing the policy of the own eventlog' must be enabled in order to modify the (remote) event log policy.

Limit the Event Log Size

  1. Open the System Settings Dialog Box if it is not open yet:
    Push the Settings button inside the Configuration panel of the System ribbon bar category.
  2. Select the Eventlogs page.
  3. Select the user account those event log files are going to be modified by selecting the appropriate computer, spacecraft and user in the associated Computer, Spacecraft and User combo boxes.
    Note:
    Connect to any host not listed by typing its name into the Computer edit field and pushing the Connect button.
  4. Select the category those event log size needs to be changed by selecting it in the Category combo box.
  5. Check one of the available file size options:

    Option

    Description

    Unlimited

    Archives events as long as space is available on the storage media.

    Limited to n KBytes

    Limits the file size to n KB.
    Enter the number n into the associated edit field.

    Use Compression

    Performs a file based compression.
  6. Repeat the last two steps if the event log size of other categories need to be changed as well.
  7. Push the OK button.

Prerequisites:
The privilege 'Creating and modifying (remote) eventlogs' or 'Changing the maximal size of the own eventlog' must be enabled in order to modify the (remote) event log size.

Use default Event Log Settings

  1. Open the System Settings Dialog Box if it is not open yet:
    Push the Settings button inside the Configuration panel of the System ribbon bar category.
  2. Select the Eventlogs page.
  3. Select the user account those event log files are going to be modified by selecting the appropriate computer, spacecraft and user in the associated Computer, Spacecraft and User combo boxes.
    Note:
    Connect to any host not listed by typing its name into the Computer edit field and pushing the Connect button.
  4. Push the Defaults button.
    Note:
    This action applies to all event log files of a user account.
  5. Push the OK button.

Prerequisites:
The privilege 'Creating and modifying eventlogs' must be enabled in order to reset the event log settings to the default values.

Reset an Event Log

  1. Open the System Settings Dialog Box if it is not open yet:
    Push the Settings button inside the Configuration panel of the System ribbon bar category.
  2. Select the Eventlogs page.
  3. Select the user account those event log files are going to be modified by selecting the appropriate computer, spacecraft and user in the associated Computer, Spacecraft and User combo boxes.
    Note:
    Connect to any host not listed by typing its name into the Computer edit field and pushing the Connect button.
  4. Select the category those event log needs to be reset by selecting it in the Category combo box.
  5. Mark the Reset check box.
  6. Repeat the last two steps if the event log of other categories need to be reset as well.
  7. Push the OK button.

Prerequisites:
The privilege 'Changing the maximal size of the own eventlog' must be enabled in order to reset an event log.

Prerequisites:
The privilege 'Creating and modifying (remote) eventlogs' must be enabled in order to modify the (remote) event log settings.
