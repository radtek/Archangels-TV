Eventbox Retrieval Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ provides a convenient way to retrieve events. Navigating through the recorded events is a simple process and very useful to track down any problems with the spacecraft or the workstation.

How do I...

Retrieve with an Eventbox

  1. Open the Eventbox to be used for the retrieval if it is not open yet.
  2. Open the Eventbox Retrievals Dialog Box by selecting the Retrievals... item on the application button's menu when the display is activated. Alternatively, push the Retrievals button on the display's toolbar.
  3. In the Retrieve from Time edit field inside the Historical Data group box specify the time at which the retrieval should start.
    The syntax for the specification of the time is: YYYY.DDD.hh:mm:ss.
  4. Specify when the retrieval should be stopped by checking one of the following options:

    Option

    Description

    Stop at YYYY.DDD.hh:mm:ss

    Stops the retrieval at the specified time. Type that time into the associated time edit field.

    Stop after n events

    Stops retrieving when the specified amount of events n is reached.

    Stop when no more events are available

    Stops retrieving events when no more data is available.
  5. Determine the playback mode by checking one of the following possibilities:

    Option

    Description

    All at once

    Retrieves all the requested events at once.

    Pseudo Realtime

    The events are replayed at a speed similar to the one when they occurred.
    Note:
    The playback mode may be changed interactively during the retrieval. Read below how to navigate through the historical events.

    Manual

    The event closest to the specified starting time is monitored and then the retrieval is stopped.
    Note:
    The playback mode may be changed interactively during the retrieval. Read below how to navigate through the historical events.
  6. Push the OK button.
Note:

Use the Global Eventbox to view the current real-time events.
