TC On-Board Buffer Queues Dump Properties Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

The TC On-Board Buffer Queues Display can help to generate and customize the telecommands necessary to dump an on-board buffer queue.

How do I...

Finalize the Dump Request

  1. Open the TC On-Board Buffer Queues Dump Dialog Box if it is not open yet:
    Select the Dump... item on the application button's menu when the display is activated. Alternatively, push the Dump button on the display's toolbar.
  2. Characterize the requested dump and push the Send... button. This opens the TC On-Board Buffer Queues Dump Properties Dialog Box.
  3. Specify the on-board execution mode by checking either the Immediately or At Time radio button. In the latter case the future on-board execution time has to be entered into the associated time spin-box.
  4. Choose the command source from which the prepared telecommand should be dispatched from by checking either the Manual Stack or Automatic Stack radio button.
  5. Decide if the telecommand has to be released immediately or at a specific time t by selecting the Immediately or At Time radio button and eventually entering the time t.
  6. Mark the Pre-Authorized check box in order to force an automatic dispatch from the chosen command source.
  7. Push the OK button.

Prerequisites:
The privilege 'Changing the telecommand on-board software services' must be enabled to configure the telecommands necessary to dump an on-board buffer queue.
