Unlock Display Window Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ provides the capability to lock/unlock display windows in order to prevent an unauthorized close action.

How do I...

Unlock a Display Window

  1. Open the Unlock Display Window Dialog Box if it is not open yet:
    Open the Windows button's menu inside the Window panel of the Home ribbon bar category and select the Lock/Unlock... item.
    Note:
    If the display window is not locked the Lock Display Window Dialog Box will appear instead.
  2. Enter the correct password into the Password entry field.
  3. Push the OK button.
Note:

Only display windows with a local scope can be locked/unlocked.
