TC Manual Stack Telecommand Groups Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

The TC Manual Stack supports groups that can be used to speed-up the search for telecommands or telecommand sequences. This dialog box is designed to provide an interface to create, modify or delete such groups of telecommands or telecommand sequences.

How do I...

Create a new Group

  1. Open the TC Manual Stack Load Telecommand Dialog Box if it is not open yet:
    Select the Load... item on the application button's menu when the display is activated. Alternatively, push the Load button on the display's toolbar.
  2. Check the TC Functions radio button.
    Note:
    Groups are not supported for telecommand sequences.
  3. Push the Groups... button. This opens the TC Manual Stack Telecommand Groups Dialog Box.
  4. Type a name for the telecommand group to be created into the Name combo box.
  5. Inside the Contents group box select the telecommand to get a member of this group inside the TC Functions combo box.
  6. Push the < button in order to add it to the group.
    Repeat the previous three steps in order to add more telecommands to a group.
  7. Push all necessary OK buttons.

Prerequisites:
The privileges 'Changing the contents of the manual telecommand stack' or 'Performing privileged tasks on the manual telecommand stack' must be enabled to create, modify or delete groups.

Modify a Group

  1. Open the TC Manual Stack Load Telecommand Dialog Box if it is not open yet:
    Select the Load... item on the application button's menu when the display is activated. Alternatively, push the Load button on the display's toolbar.
  2. Check the TC Functions radio button.
  3. Push the Groups... button. This opens the TC Manual Stack Telecommand Groups Dialog Box.
  4. Select the name of the group to be modified inside the Name combo box. All telecommands belonging to the chosen groups are displayed within the Members list box.
  5. Perform the required changes:
    • Select a telecommand to be added inside the TC Functions combo box and push the < button.
    • Choose the member to deleted inside the Members group and push the > button. The >> button clears the complete group.
  6. Push all necessary OK buttons.

Prerequisites:
The privileges 'Changing the contents of the manual telecommand stack' or 'Performing privileged tasks on the manual telecommand stack' must be enabled to create, modify or delete groups.

Delete a Group

  1. Open the TC Manual Stack Load Telecommand Dialog Box if it is not open yet:
    Select the Load... item on the application button's menu when the display is activated. Alternatively, push the Load button on the display's toolbar.
  2. Check the TC Functions radio button.
  3. Push the Groups... button. This opens the TC Manual Stack Telecommand Groups Dialog Box.
  4. Select the group to be deleted inside the Name combo box.
  5. Delete it by pushing the Remove button. The Remove All button clears all groups at once.
  6. Push all necessary OK buttons.

Prerequisites:
The privileges 'Changing the contents of the manual telecommand stack' or 'Performing privileged tasks on the manual telecommand stack' must be enabled to create, modify or delete groups.
