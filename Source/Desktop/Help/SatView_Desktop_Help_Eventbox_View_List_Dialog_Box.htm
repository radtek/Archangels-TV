Scheduled Event List Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ uses events as standard user notification feature. Events can be scheduled to occur at a certain time. Due to the fact that many actions like for example playing sound, voice alerts or notifications by a phone call can be attached to an event almost any kind of action can be initiated at a certain time.

How do I...

View a scheduled Event

  1. Open the Scheduled Event List Dialog Box if it is not open yet:
    Activate the Global Eventbox and then select the Watch... item on the application button's menu. Alternatively, push the Watch button on the display's toolbar.
  2. Select the scheduled event to be viewed in the List list box.
  3. Push the Details... button to make the Event Details Dialog Box appear where all details about the scheduled event can be obtained.
  4. Push the OK button.

Prerequisites:
The privilege 'Watching the scheduled event list of the global eventbox display window' must be enabled in order to watch the list of scheduled events in the Global Eventbox.

Modify a scheduled Event

  1. Open the Scheduled Event List Dialog Box if it is not open yet:
    Activate the Global Eventbox and then select the Watch... item on the application button's menu. Alternatively, push the Watch button on the display's toolbar.
  2. Select the scheduled event to be modified in the List list box.
  3. Push the Change... button to make the Event Schedule Dialog Box appear.
  4. Perform all necessary changes as described in 'Scheduling an Event'.
  5. Push the OK button.

Prerequisites:
The privileges 'Watching the scheduled event list of the global eventbox display window' and 'Modifying the scheduled event list of the global eventbox display window' must be enabled in order to modify scheduled events in the Global Eventbox.

Delete a scheduled Event

  1. Open the Scheduled Event List Dialog Box if it is not open yet:
    Activate the Global Eventbox and then select the Watch... item on the application button's menu. Alternatively, push the Watch button on the display's toolbar.
  2. Select the scheduled event to be deleted in the List list box.
  3. Push the Remove button to delete the selected event from the scheduled list.
    Use the Remove All button to empty the list.
  4. Push the OK button.

Prerequisites:
The privileges 'Watching the scheduled event list of the global eventbox display window' and 'Modifying the scheduled event list of the global eventbox display window' must be enabled in order to delete scheduled events in the Global Eventbox.
