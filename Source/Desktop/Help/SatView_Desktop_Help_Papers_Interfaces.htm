  • Packet Front-End Link Protocol (PFLP)
  • SatView™ supports the Packet Front-End Link Protocol (PFLP) as defined by ESA and uses it to communicate with telemetry data provider devices. This document describes the various implementations of PFLP necessary to interface with the supported front-end equipment.

  • Telemetry Parameter Exchange Protocol (TPEP)
  • SatView™ incorporates a custom protocol supporting the distribution of selected telemetry parameters, source packets or formats. It allows to request real-time, historical or random (for testing purposes) telemetry data in various formats.

  • Memory Dumps Interface
  • SatView™ comes with a special memory dump interface that allows the processing, archiving and distribution of on-board memory dumps alongside the standard telemetry data services.
