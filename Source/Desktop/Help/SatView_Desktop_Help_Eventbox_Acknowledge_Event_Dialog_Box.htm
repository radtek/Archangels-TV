Event Acknowledge Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ uses events as standard user notification feature. Due to this those events sometimes need to be acknowledged.

How do I...

Acknowledge an Event

  1. Open the Event Acknowledge Dialog Box:
    Activate the Eventbox and then select the Acknowledge... item on the application button's menu. Alternatively, push the Acknowledge button on the display's toolbar.
  2. Push the Update button if not all events that need to be acknowledged are contained in the List list box.
  3. Choose the following filtering options if not all alerted events should be acknowledged:

    Option

    Description

    By Category

    Mark the System, Spacecraft and/or User check box. This will acknowledge only those events belonging to the selected categories.

    By Type

    Check the associated Restricted to Type radio button and select the type of event to be acknowledged.

    By User

    Check the associated Restricted to User radio button and select the user those events should be acknowledged.

    By Time

    Mark the associated From t1 to t2 check box and fill in the time period within which all events should be acknowledged.
  4. Push the Update button in order to verify in the List list box which events are going to be acknowledged.
  5. Push the OK button.
Note:

A single event can also be acknowledged with the Eventbox Details Dialog Box.

Prerequisites:
The privilege 'Acknowledging events in the global eventbox display window' must be enabled in order to acknowledge events in the Global Eventbox.
