TC Manual Stack Telecommand Block Macro Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

The TC Manual Stack provides the necessary support to interactively block several telecommands into a telecommand packet which can be done with this dialog box.

How do I...

Block Telecommands

  1. Open the TC Manual Stack Telecommand Properties Dialog Box or TC Manual Stack Telecommand Sequence Properties Dialog Box if it is not open yet:
    Activate the TC Manual Stack, select the telecommand or telecommand sequence those properties need to be viewed or modified and select the Edit è Properties... item on the application button's menu. Alternatively, push the Properties button on the display's toolbar.
    Note:
    If the telecommand or telecommand sequence is not yet loaded onto the stack proceed as follows:
    1. Open the TC Manual Stack Load Telecommand Dialog Box if it is not open yet:
      Select the Load.. item on the application button's menu when the display is activated. Alternatively, push the Load button on the display's toolbar.
    2. Check the TC Functions radio button to load a telecommand or the TC Sequences radio button for telecommand sequences and select the desired item inside the list box below the Filter combo box or type its name into the Load edit field.
    3. Push the Properties... button.
      When a telecommand sequence is loaded, select the required member inside the Entry combo box.

    Important: It must be made sure that only a telecommand able to become a 'block head' is selected.

  2. Push the Release... button. The TC Manual Stack Telecommand Release Constraints Dialog Box will pop-up.
  3. Mark the Blocked check box and push the Macro... button.
    This opens the TC Manual Stack Telecommand Block Macro Dialog Box:

    Adding a new Block Macro Member

    1. Select the telecommand to be added to the macro inside the Members combo box.
    2. Activate the position in the Macro list box after which the selected telecommand should be inserted and push the < button.
      Repeat the previous steps if more than one member should be added.
      Note:
      The required size is always recalculated after modifying a block macro and, in case the resulting block is too big, a warning is issued.

    Deleting a Block Macro Member

    1. Select the telecommand to be removed from the macro inside the Members combo box.
    2. Push the > button.
      Repeat the previous steps if more than one member should be deleted or use >> in order to clear the complete contents of the block macro.
      Note:
      The head telecommand of a block macro can never be deleted.
  4. Push all necessary OK buttons.

Prerequisites:
The privileges 'Changing the contents of the manual telecommand stack' or 'Performing privileged tasks on the manual telecommand stack' must be enabled to configure a telecommand.
