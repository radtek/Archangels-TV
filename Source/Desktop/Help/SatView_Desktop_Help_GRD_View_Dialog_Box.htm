GRD Parameters Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

Graphic Displays (GRD) provide the capability to display additional details about the telemetry parameters they contain.

How do I...

View GRD Parameter Details

  1. Open the GRD Parameters Dialog Box if it is not open yet by double-clicking on the parameter inside the GRD to be viewed or by selecting the Parameters... item on the application button's menu when the display is activated. Alternatively, push the Telemetry Parameters button on the display's toolbar.
  2. Select the parameter those properties should be shown in the Parameter combo box of the dialog box (if necessary).
  3. Choose the appropriate property page in order to view the desired information.
  4. Push the OK button.
Note:

Additional parameter details can also be obtained through the use of tool-tips: Right-click on one of the parameters to view additional column related information.

Scale GRD Parameters

  1. Open the GRD Parameters Dialog Box if it is not open yet by double-clicking on the parameter inside the GRD to be scaled or by selecting the Parameters... item on the application button's menu when the display is activated. Alternatively, push the Telemetry Parameters button on the display's toolbar.
  2. Select the parameter those properties should be shown in the Parameter combo box of the dialog box (if necessary).
  3. Choose the Scaling property page.
  4. Enter the required minimum and maximum values into the corresponding edit fields located to the right of the current range.
    Note:
    • Push the Refresh button in order to view the current minimum and maximum values of the selected parameter.
    • Push the Reset button in order to restore to the previous range or the Default button to reset the range to the database defaults.
    • Mark the Upper and/or Lower check box and enter a percentage into the associated spin box in order include a margin to the top or bottom of the plotting area.
  5. If a scaling should occur automatically, mark the Autoscaling check box and enter the number of minutes into the associated spin box after which a scaling should occur.
  6. Push the OK button.

Zoom GRD Parameters

  1. Push the left mouse button and the 'Shift' key at the starting position (on the graph) to be zoomed-in. A ruler will occur.
  2. Move the mouse to the right by keeping the left button and 'Shift' key pressed. A small tool-tip above the ruler will indicate the time range to be activated.
    Note:
    Push the 'Ctrl' key in order to activate the ruler.
  3. Release the mouse button.
Note:

The graphic display will be set to hold mode after zooming-in whenever the time range is completely within historical telemetry data. Pushing the Hold button again will restore to normal update mode.
