SatView™ uses a Database Management System (DBMS) to maintain most parts of a database. Depending on the DBMS used for a certain database, different settings and optimization issues need to be configured.

How do I...

Configure & Optimize the DBMS Settings

  1. Open the DBMS Settings Dialog Box if it is not open yet:
    Push the Details... button of the New Dialog Box when the database is going to be created.
  2. Check the Data radio button in order to indicate that all following information is related to the data file of the database.
  3. Enter the name of the file into the Filename edit field.
  4. Specify the directory where the file should be located by typing it into the Location edit field. Use the Browse... button in order to find a suitable directory.
  5. Determine the initial, maximal and grow size of the file by entering the various sizes (in MB) into the associated Initial Size, Maximal Size and Grow Size edit fields.
    Note:
    Mark the Unlimited check box if a maximal size for the file is not applicable. Use the Unchangeable check box if the selected database file should be limited in its growth.
    The Read-only check box can be marked if the database should be 'read-only'.
  6. Check the Logs radio button and repeat the last steps (except for the first one) in order to specify the same information for the logs file of the database.
  7. Push the OK button.

Prerequisites:
The privileges 'Creating databases' or 'Creating (remote) databases' must be enabled to change the (remote) DBMS settings.
