MMD Properties Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ provides an interface to customize the properties for a mimics display which not only include the image size, margins or background but also elements aiding the design of the display that might not be visible anymore afterwards.

How do I...

Configure the Properties of a Mimics Display

  1. Open the MMD Properties Dialog Box if it is not open yet:
    Select the Properties button inside the Details panel of the View ribbon bar category.
  2. Check either the Default or Fixed radio button depending on the requirement to have an image with a custom size (no automatic adjustment) or not.
    For an image with a custom size type the desired width and height (in pixels) into the corresponding edit fields.
  3. Enter the size (in pixels) of a required margin into the Left, Right, Top and/or Bottom edit field.
    Note:
    An image with an automatic size adjustment cannot have a left or top margin.
  4. Choose the color to be the background of the display in the Background combo box.
  5. Check either the Points or Lines radio button depending on whether points or lines should be added to the background. If this option is enabled enter the horizontal and vertical interval (in pixels) between them into the Horizonal Interval or Vertical Interval edit field and choose their color in the Color combo box.
    Note:
    This option is available primarily for design purposes. By marking the Show always button the grid can be made visible even after that time.
  6. Push the OK button.

Prerequisites:
The privilege 'Creating and modifying mimics displays' must be enabled to change the properties of mimics displays.
