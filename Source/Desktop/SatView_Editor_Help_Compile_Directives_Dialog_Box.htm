Compile Directives Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ uses third-party software to generate executable modules from certain database component items. In some situations it might be necessary to (temporarily) suppress messages issued by these applications. This can be achieved by creating compile directives profiles.

How do I...

Create a new Compile Directives Profile

  1. Compile the database components first before creating or changing the compile directives.
    The resulting messages will appear in the Build tab of the Output tool window.
  2. Open the Compile Directives Dialog Box if it is not open yet:
    Select the Build ribbon bar category and push the Directives button inside the Settings panel.
  3. Enter the name of the directives profile to be created into the Profile edit field.
  4. Select those messages in the Messages list box that need to be suppressed in future compilations when this profile is applied and then push the Disable button.
    Note:
    When messages need to be re-enabled they must be selected in the Disabled list box first before using the Enable button.
    Use the Delete button to remove a disabled message which does not need to be re-enabled any more because the cause for its occurrence has been resolved.
    Mark the Do not show any warnings check box if no warnings should be listed when a database is compiled using the selected compile directives profile.
  5. Push the Add button.
    Repeat the last three steps in order to add more than one compile directives profile.
  6. Push the OK button.

Delete a Compile Directives Profile

  1. Open the Compile Directives Dialog Box if it is not open yet:
    Select the Build ribbon bar category and push the Directives button inside the Settings panel.
  2. Select the name of the directives profile to be deleted inside the Profile combo box.
  3. Push the Remove button.
    Use the Remove All button in order to delete all compile directives profiles.
    Note:
    The Default compile directives profile cannot be deleted.
  4. Push the OK button.

Apply a Directives Profile to a Compile Pass

  1. Open the Compile Directives Dialog Box if it is not open yet:
    Select the Build ribbon bar category and push the Directives button inside the Settings panel.
  2. Select the name of the directives profile to be applied to the next compilation pass inside the Profile combo box.
  3. Push the OK button.
