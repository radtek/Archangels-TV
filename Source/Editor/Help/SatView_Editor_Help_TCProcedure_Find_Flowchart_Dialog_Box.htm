TC Procedure Find Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ supports an interface to find flow-control steps and action-object steps in telecommand procedures.

How do I...

Find a Step

  1. Open the Find Dialog Box if it is not open yet: Move the focus to the flowchart pane of the thread to be searched and push the Find button inside the Editing panel of the Home ribbon bar category.
  2. Enter the name of the step to be searched for into the Find Step edit field.
    Note:
    Mark the Search hidden names check box if also steps should be searched those (temporary) name is automatically generated (and invisible on the flowchart view).
  3. Push the Find button to start the search.
    If the requested step is found it is activated.
  4. Push the Close button if the search was not successful.
