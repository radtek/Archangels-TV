Arc Properties Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

The Arc mimics object renders as a part of a circle or an ellipse. By changing its outline it can be transformed into a pie or chord.

How do I...

Change the Properties

  1. Select the General page.
  2. Type the name of the mimics object into the Name edit field.
    Note:
    The string 'No Name' indicates that the mimics object has no name which is recommended for all objects that do not have any associated code.
  3. Enter any text into the Tooltip edit field if a tool-tip should be attached to the mimics object.
    Note:
    This tool-tip will become visible when right-clicking onto that object (in the operational environment only).
  4. Select the Outline page.
  5. Choose between the available shapes:

    Shape

    Description

    Arc

    Draws the border of a circular segment.

    Pie

    Draws a shaped wedge bounded by the intersection of an ellipse and two radials.

    Chord

    Draws a region bounded by the intersection of an ellipse and a line segment, called a secant.
    Note:

    Use the gray thumbs to adjust the arc in size or the yellow ones to determine how much of the circle or ellipse should be visible.

  6. Inside the Border group box choose the border style in the Style combo box, type the border size (in pixels) into the Size edit field and select the border color in the Color combo box.
  7. Within the Interior group box select the interior color in the Color combo box, choose the interior hatch in the Hatch combo box, the interior hatch color in the Color combo box below the selected hatch and mark the Transparent check box if the interior should be transparent.
    Note:
    Interior properties are available for pies and chords and when drawn in a non transparent way only.
  8. Push the OK button.

Prerequisites:
The privilege 'Creating and modifying mimics displays' must be enabled to edit the properties of mimics objects.
