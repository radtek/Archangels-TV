TC Procedure Test Profile Parameters Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

This dialog box provides an interface to choose telemetry parameters (initialized with a definable value) to be included into a test profile.

How do I...

Choose Test Profile Parameters

  1. Open the TC Procedure Test Profile Parameters Dialog Box if it is not open yet:
    Select the Debug property page, choose the appropriate thread in the Thread combo box and push the Add button inside the Properties group box.
  2. Select a telemetry parameter inside the Parameter combo box.
  3. Specify how the value that is going to be supplied in the next step is to be interpreted:

    Calibration

    Description

    Raw

    The supplied value will not be calibrated.

    Calibrated

    The value will be in a calibrated form whereby the calibration takes place with the table indicated in the Table combo box.
    The desired calibration curve can be selected in the Table combo box if the calibration type is not 'raw'.
    The coding is indicated in the Coding combo box and is fixed.

    Radix

    Description

    Binary

    The supplied value will be expressed with a binary radix.

    Octal

    The value will be in a supplied with an octal radix.

    Decimal

    The value will be in a supplied with a decimal radix.

    Hexadecimal

    The value will be in a supplied with a hexadecimal radix.
    Note:

    The radix can only be specified or changed when a raw value with a coding other that 'floating-point' is going to be supplied.

  4. Enter a value (formatted as specified previously) into the Value edit field.
    Note:
    Make sure to enter a value corresponding with the specified calibration type, coding and radix. In particular, any radix other than decimal will require a constant (with the specified radix) to be specified and not an expression.
  5. Push the Add button in order to insert the current telemetry parameter into the test profile. Repeat the previous steps (except for the first one) as many times as parameters need to be added to the test profile.
    Use the Remove or Remove All button in order to delete a single or all parameters from the list.
  6. Push the OK button.
