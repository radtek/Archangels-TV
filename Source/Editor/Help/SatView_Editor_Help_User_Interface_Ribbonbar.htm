Ribbon Bar

SUMMARY

Saturday, October 10, 2015
Version 1.0

The ribbon bar contains a Quick Access Toolbar (QAT) that can be populated with commands that are frequently used.

How do I...

Add commands to the QAT

  1. Open the User Interface Dialog Box if it is not open yet:
    Select the menu item: More Commands... in the QAT menu.
  2. Select the Ribbon Bar page.
  3. Select the category in the Choose commands from combo box to which the command to be added to the QAT is belonging to.
  4. Choose the command to be added in the Commands list box.
  5. Push the Add button.
    Repeat the previous two steps if more than one command should be added.
  6. Push the OK button.

Remove commands from the QAT

  1. Open the User Interface Dialog Box if it is not open yet:
    Select the menu item: More Commands... in the QAT menu.
  2. Select the Ribbon Bar page.
  3. Select the category in the Choose commands from combo box to which the command to be removed from the QAT is belonging to.
  4. Choose the command to be deleted in the right list box.
  5. Push the Remove button.
    Repeat the previous two steps if more than one command should be deleted.
    Note:
    Push the Reset button in order to remove all commands from the QAT.
  6. Push the OK button.

Change the command sequence order on the QAT

  1. Open the User Interface Dialog Box if it is not open yet:
    Select the menu item: More Commands... in the QAT menu.
  2. Select the Ribbon Bar page.
  3. Choose the command those position on the QAT should be changed in the right list box.
  4. Push either the Up or Down button depending on whether the command should be moved up or down.
    Repeat the previous two steps until the desired command sequence order is achieved.
    Note:
    Shifting a command upwards moves it to the left on the QAT.
  5. Push the OK button.

Change the position of the QAT

  1. Open the User Interface Dialog Box if it is not open yet:
    Select the menu item: More Commands... in the QAT menu.
  2. Select the Ribbon Bar page.
  3. Mark the Show Quick Access Toolbar below the Ribbon check box if the QAT should be displayed on a separate pane below the ribbon bar.
  4. Push the OK button.
