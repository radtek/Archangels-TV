Control-Thread Step Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

This action-object step controls the execution state of the specified thread inside the current telecommand procedure.

How do I...

Configure the Control-Thread Step

  1. Open the Control-Thread Step Dialog Box if it is not open yet:
    Double-click onto the step to be configured or activate it and select the Properties button on the toolbar.
  2. Select the thread those execution state should be changed in the Name combo box.
  3. Choose the execution state to be applied to the selected thread:

    State

    Description

    Suspend

    Suspends the execution of the specified thread.

    Resume

    Resumes the execution of the specified (suspended) thread.

    Enable

    Enables the specified thread.
    Note:
    Only an enabled thread can start its execution (even if its activation expression is true).

    Disable

    Disables the specified thread.
    Note:
    A thread can be disabled during its execution. When a disabled thread terminates it re-enters and waits to be re-enabled as well as for its activation expression to become true again.

    Abort

    Aborts the execution of the specified thread.
  4. Push the OK button.

Prerequisites:
The privilege 'Creating and modifying telecommand procedures' must be enabled to modify telecommand procedures.
