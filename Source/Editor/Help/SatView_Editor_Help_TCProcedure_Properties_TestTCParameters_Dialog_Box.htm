TC Procedure Test Parameters Details Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

This dialog box simplifies the determination of the correct value for telecommand parameters with an attached calibration table by showing all points of that table.

How do I...

Browse Test Parameter Details

  1. Open the TC Procedure Test Parameters Details Dialog Box if it is not open yet:
    Select the Debug property page, make sure that the telecommand procedure is in a suspended state and double-click the telecommand parameter those calibration details should be shown inside the Watch list box.
    Note:
    This feature is available only for telecommand parameters that have an associated calibration curve.
  2. Browse the calibration table details on the Calibration page.
    Note:
    If a predefined calibrated point is appropriate to be used as a value for the telecommand parameter, that value can directly be copied to the Value edit field by double-clicking on it.
  3. Push all necessary OK buttons.
