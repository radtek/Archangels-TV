This action-object step requests (predefined) values for variables from the operator.

How do I...

Configure the User-Interaction Step

  1. Open the User-Interaction Step Dialog Box if it is not open yet:
    Double-click onto the step to be configured or activate it and select the Properties button on the toolbar.
  2. Choose the required interaction type in the Type combo box:

    Type

    Description

    Acknowledge only

    Requires the operator to just acknowledge the message specified in the Message edit field.

    Request a value for variables

    Asks the operator to enter a value for all the variables listed in the Selected Variables list box.

    Choose a predefined value for variables

    Asks the operator to choose a value out of a predefined list of values for all the variables listed in the Selected Variables list box.
  3. Enter the message in the Message edit field.
  4. Select the variables to be set by the operator in the Available Variables and push the > button.
    Use the < button to remove any variables from the Selected Variables list box.
    Note:
    This step only applies to the types 'Request a value for variables' and 'Choose a predefined value for variables'.
  5. Enter a value in the Value edit field that should be added to the predefined set of values for a variable selected in the Selected Variables and push the > button.
    Use the < button to remove the selected value from the set or << to delete all values.
    Note:
    This step only applies to the type 'Choose a predefined value for variables'.
  6. Push the OK button.

Prerequisites:
The privilege 'Creating and modifying telecommand procedures' must be enabled to modify telecommand procedures.
