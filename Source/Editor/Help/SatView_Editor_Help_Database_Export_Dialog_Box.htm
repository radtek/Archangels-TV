Database Export Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ provides an interface to export tables from a database to files. This is useful to create backups or to synchronize databases.

How do I...

Export Databases

  1. Open the Database Export Dialog Box if it is not open yet:
    Select the Export Database... item on the application button's menu.
  2. Choose the server from which the database to be exported is located by selecting it in the Server combo box.
  3. Determine the spacecraft to which the database belongs to in the Spacecraft combo box.
  4. Select one of the databases available for the desired spacecraft on the specified server by highlighting it in the Database combo box.
  5. Mark the Export all tables check box if all tables of the selected database should be exported to files or check Only if a subset of tables should be exported. In the latter case, select a table in the right list box and push the < button in order to add it to the list of tables to be exported. Repeat the last step as many times as tables need to be exported.
    Use the > button to remove a selected table inside the left list box (foreseen to be exported) or push the >> to delete all of them.
  6. Enter the destination directory of the export files into the Location edit field.
    Note:
    Use the Browse... button to search for an existing directory.
  7. Type a file name prefix into the Prefix edit field if the resulting file names should be built with this prefix. In addition, it is recommended to enter a file extension into the Extension edit field.
    The file name is built as follows: Location\PrefixTable_Name.Extension
    Note:
    The prefix and extension can be left empty.
  8. Inside the Format group box choose the string, field and row delimiters to be used when the database table is exported. Just select the appropriate items in the String Delimiter, Field and Row combo boxes.
  9. Push the Unicode check box if the exported file should be encoded in Unicode (and not as an ASCII file).
  10. Push the OK button.
