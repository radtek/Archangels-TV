Programming Languages

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ provides some custom programming languages supporting the definition of Derived Parameters, the design of Mimics Displays and the coding of Telecommand Procedures. They are all accompanied by a common library of functions granting simple access to any kind of telemetry data or telecommands.

  • Derived Parameter Language (DPL)
  • The Derived Parameter Language (DPL) is the programming language of SatView™ developed to ease the implementation of Derived Parameters. It encapsulates the code for each derived parameter in a separate procedure. Special DPL keywords are available to control the computation of those parameters.
  • January 2016
  • 1.06
  • Mimics Description Language (MDL)
  • The Mimics Description Language (MDL) is the programming language of SatView™ developed to ease the description of the dynamic behavior of Mimics Displays (MMD). The animation of mimics objects is defined through individual procedures for each of those. The MDL supports an extended set of keywords (compared to the DPL) in order to control the execution of the procedures.
  • January 2016
  • 1.07
  • Telecommand Procedure Language (TPL)
  • The Telecommand Procedure Language (TPL) is the programming language of SatView™ to implement Telecommand Procedures. The usual way to design those procedures is with the supported interactive graphical interface provided by the SatView™ Editor which automatically converts the procedure into TPL code. However, it is also possible to write such code directly.
