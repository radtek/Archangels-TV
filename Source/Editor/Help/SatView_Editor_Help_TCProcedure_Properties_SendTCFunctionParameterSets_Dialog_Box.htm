TC Procedure Parameter Sets Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

Some of the telecommand procedure steps are designed to sent telecommands those parameters may need to be initialized with values. This dialog box shows the predefined parameter value sets available for a selected telecommand.

How do I...

Choose a Telecommand Parameter Set

  1. Open the TC Procedure Parameter Sets Dialog Box if it is not open yet:
    Open the properties of the corresponding Send-TC Function step or the Send-TC Sequence step and push the Load Values... button.
  2. Select one of the predefined value sets to be used for the initialization of the telecommand parameters in the Name combo box.
    Note:
    The contents of the selected value set is displayed in the Contents list box. If a telecommand contains more parameters than listed by the selected set only those will be initialized that are also member of the parameter value set.
  3. Push the OK button.
