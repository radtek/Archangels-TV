Interfaces:

  • Packet Front-End Link Protocol (PFLP)
     
     
     
  • Telemetry Parameter Exchange Protocol (TPEP)
     
     
     
  • Memory Dumps Interface

 

  • Describes the current implementation of the Packet Front-End Link Protocol (PFLP). At the moment, its use is limited to the communication with telemetry data provider equipment.
  • Describes the definition of the Telemetry Parameter Exchange Protocol (TPEP). This protocol is suited for distributing real-time or historical telemetry parameter values in a heterogeneous environment.
  • Describes the definition of the Memory Dumps Interface. This interface has been implemented to provide a light-weight solution for the distribution of (processed) on-board memory dumps.
