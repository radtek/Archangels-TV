SatView™ supports an interface to find and replace graphic display definition related xml code.

How do I...

Jump to a Code Line

  1. Open the Goto Dialog Box if it is not open yet: Move the focus to the code pane and push the Go To button inside the Editing panel of the Home ribbon bar category.
  2. Choose one of the available options:

    Option

    Description

    Number: n

    Jumps to line n and highlights it.

    Jump down: n Lines

    Jumps n lines down (if possible) and highlights that one.

    Jump up: n Lines

    Jumps n lines up (if possible) and highlights that one.
  3. Push the Go button. This performs the required jump.
  4. Push the OK button.
