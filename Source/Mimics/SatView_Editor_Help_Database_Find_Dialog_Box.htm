Database Find Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ provides an interface with which a database can be searched for parameters, displays and other component items. The most common queries are contained in the corresponding pages. Complex searches can be executed by using the DBMS language directly.

How do I...

Search a Database

  1. Open the Database Find Dialog Box if it is not open yet:
    Push the Find button inside the Editing panel of the Home ribbon bar category.
    Note:
    Make sure that a form view (used to edit core DBMS component items) is the active display window before pushing the button.
  2. Select the page related to the database component to be searched:

    Component

    Description

    TM Packets

    Searches all telemetry packets.

    TC Functions

    Searches all telecommand functions.

    TM Parameters

    Searches all telemetry parameters.

    TC Parameters

    Searches all telecommand parameters.

    On-Board Processors

    Searches all on-board processors.

    ANDs

    Searches all alphanumeric displays.

    GRDs

    Searches all graphic displays.

    PODs

    Searches all parameter observation displays.

    DBMS Query

    Searches the DBMS by providing an interface to specify SQL statements directly.
    Note:

    The number of pages may vary depending on the standard of the database searched.

  3. Specify the search criteria. If multiple criteria are specified, they will be combined through a logical 'AND'.
  4. Check either the Pane 1 or Pane 2 radio button.
    The results will then be directed to the Find I or Find II tab of the Output tool window.
  5. Push the OK button.
Note:

The above search interface is available for core DBMS database components only. Those of them that are not part of the core DBMS provide different search interfaces. Most of them have associated source code that can be searched, sometimes even the capability to search for graphical elements.
Refer to the individual chapters about those database components to learn more how to search them.
