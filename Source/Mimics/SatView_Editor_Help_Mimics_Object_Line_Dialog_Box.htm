Line Properties Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

The Line mimics object connects two points in the shortest possible way. By changing its outline it can be transformed into a (double) arrow and, on demand, symbolize a crossing.

How do I...

Change the Properties

  1. Select the General page.
  2. Type the name of the mimics object into the Name edit field.
    Note:
    The string 'No Name' indicates that the mimics object has no name which is recommended for all objects that do not have any associated code.
  3. Enter any text into the Tooltip edit field if a tool-tip should be attached to the mimics object.
    Note:
    This tool-tip will become visible when right-clicking onto that object (in the operational environment only).
  4. Select the Outline page.
  5. Choose the line style in the Style combo box and the desired color in the Color combo box.
  6. Type the line thickness (in pixels) into the Thickness edit field.
  7. Mark the Crossing check box if such a symbol should be drawn on the line.
    Note:
    Use the yellow thumb to position the crossing symbol on the line.
  8. Mark the Arrow or Double Arrow check box if a single-sided/double-sided arrow should be drawn.
  9. Push the OK button.

Prerequisites:
The privilege 'Creating and modifying mimics displays' must be enabled to edit the properties of mimics objects.
